9. Tunjukkan bahwa persamaan garis singgung parabola x² = 4py yang melalui titik (x₁, y₁) pada parabola adalah x₁x = 2p(y + y₁) !


Irisan Kerucut.

Dengan menggunakan konsep turunan dimana m = dy/dx, untuk mengerjakan soal ini gunakan turunan implisit!
dy/dx (x²) = dy/dx (4py)
2x = 4p dy/dx
dy/dx = x / (2p) → m = x / (2p)

y - y₁ = m(x - x₁)
y - y₁ = x / (2p) (x - x₁)
y - y₁ = x² / (2p) - x₁x / (2p)
y - y₁ = (4py - x₁x) / (2p)
4py - x₁x = 2p(y - y₁)
-x₁x = -2py - 2py₁
x₁x = 2p(y + y₁)
Tertunjuk.
