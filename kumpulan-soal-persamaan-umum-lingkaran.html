8. tentukan persamaan lingakaran yang berpusat di titik (-2,-4) dan menyinggung garis 8x+15y+25=0 ! (nyatakan dalam bentuk umum persamaan lingkaran)tolong dijawab dan caranya


Tentukan persamaan lingakaran yang berpusat di titik (-2, -4) dan menyinggung garis 8x + 15y + 25 = 0 ! (nyatakan dalam bentuk umum persamaan lingkaran).

Persamaan Lingkaran adalah tempat kedudukan titik-titik (x, y) yang berjarak sama terhadap satu titik tertentu.

Persamaan umum lingkaran

Berpusat di pangkal koordinat

        x² + y² = r²

Berpusat di titik (a, b)

        (x - a)² + (y - b)² = r²

Pembahasan

Diketahui:

Pusat lingkaran = (-2, -4)

Menyinggung garis 8x + 15y + 25 = 0

Ditanya:

Persamaan Lingkarannya

Jawab:

Kita cari jari-jarinya terlebih dahulu

Dari persamaan 8x + 15y + 25 = 0

a = 8

b = 15

c = 25

Dari pusat lingkaran (-2, -4)

x₁ = -2

y₁ = -4

Jari-jari = I [tex]\frac{ax_{1+by_{1}+c} }{\sqrt{a^2+b^2} }[/tex] I

            = I [tex]\frac{8.-2+15.-4+25}{\sqrt{8^2+15^2} }[/tex] I

            = I [tex]\frac{-16-60+25}{\sqrt{64+225} }[/tex]I

            = I[tex]\frac{-51}{17}[/tex] I

            = I -3 I

            = 3

Kita buat persamaan lingkarannya

Pusat ( a, b)

(x - a)² + (y - b)² = r²

(x - (-2))² + (y - (-4))² = 3²

(x + 2)² + (y + 4)² = 3²

x² + 4x + 4 + y² + 8y + 16 = 9

x² + y² + 4x + 8y + 20 - 9 = 0

x² + y² + 4x + 8y + 11 = 0

Jadi Persamaan lingkarannya dalam bentuk persamaan umum lingkaran adalah (x + 2)² + (y + 4)² = 3² atau x² + y² + 4x + 8y + 11 = 0.
