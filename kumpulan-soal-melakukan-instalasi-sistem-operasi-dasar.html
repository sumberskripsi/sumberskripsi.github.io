》Perbedaan Instalasi Sistem Operasi Closed Source dan Open Source

Sistem Operasi Close Source adalah Sistem operasi yang kodenya tidak dibuka untuk umum. Pemilik kode close source bisa membagikan source codenya melalui lisensi secara gratis atau membayar.

Pada sistem ini paket program tidak dapat didistribusikan lagi selain oleh Pembuat program tersebut.

Sedangkan Sistem Operasi Open Source adalah sistem operasi yang kodenya dibuka untuk umum dan dapat dikonsumsi oleh masyarakat melalui lisensi gratis.

14. tata cara melakukan partisi hardisk pada saat proses instalasi sistem operasi.​


Penjelasan:

Pada saat instalasi sistem operasi, mempartisi harddisk adalah hal yang penting guna memberikan lokasi dimana sistem operasi akan ditempatkan

Tata cara:

pada saat instalasi, biasanya akan muncul tampilan untuk mempartisi harddisk, (untuk instalasi sistem operasi windows 7/8/8.1/10)

- Pertama klik harddisk yang akan di partisi

- lalu klik "New" pada bagian bawah tampilan

- lalu akan muncul "size" / ukuran partisi yang akan kamu buat untuk sistem operasi nya, masukan sesuai yang kamu inginkan (misal 250GB berarti masukkan 250000 MB) lalu klik ok

- kemudian kita buat lagi partisi kedua, dengan mengklik lagi harddisk yang berlabel "Unallocated Space" dan klik new dan masukkan ukuran partisi yang ingin kamu buat

selesai deh, tahap terakhir:

- Klik pada partisi yang pertama kali kamu buat (partisi dengan type primary) lalu klik next di pojok kanan bawah, maka sistem operasi akan di install pada partisi yang kamu pilih / yang pertama kamu buat

21. Instalasi sistem operasi hal yang harus dilakukan adalah setting BIOS, apa saja yang harus di setting pada BIOS!​


Jawaban:

Untuk melakukan instalasi sistem operasi, ada beberapa pengaturan BIOS yang perlu diubah. Berikut adalah beberapa hal yang perlu kamu setting pada BIOS:

Boot Priority: Pastikan boot priority atau urutan boot diatur sedemikian rupa sehingga drive tempat instalasi sistem operasi berada menjadi prioritas pertama untuk memulai.    Secure Boot: Jika menggunakan versi Windows terbaru, pastikan Secure Boot dinonaktifkan agar sistem dapat mengenali tanda tangan digital dari perangkat lunak yang akan diinstal.    Legacy Mode/UEFI Mode: Pilih mode legacy atau UEFI sesuai dengan kebutuhan dan kompatibilitas hardware serta software kamu.    SATA Mode/AHCI: Ubah mode SATA menjadi AHCI jika ingin menggunakan fitur-fitur canggih seperti hot swapping dan NCQ pada hard disk drive (HDD) atau solid-state drive (SSD).    Time and Date Settings: Pastikan waktu dan tanggal yang ditampilkan oleh BIOS sudah benar agar tidak menyebabkan masalah saat instalasi dan konfigurasi sistem operasi nantinya.    Save and Exit Changes: Setelah mengubah pengaturan-pengaturan tersebut, jangan lupa simpan perubahan sebelum keluar dari BIOS.

26. 1. alat dan bahan apa saja yg dibutuhkan dalam melakukan instalasi sistem operasi?2. uraikan dengan detail langkah langkah instalasi sistem operasi windows 10!​


Alat dan bahan yang dibutuhkan dalam melakukan instalasi sistem operasi adalah file installer bisa dalam bentuk .iso atau file installer lainnya. Agar file installer dapat terbaca oleh komputer, file installer perlu dipindah ke flashdisk atau DVD.

Langkah-langkah instalasi sistem operasi windows 10 adalah sebagai berikut

Atur bios untuk booting pertama pada flashdisk atau DVD room.Restart komputer atau laptop, pastikan flashdisk installer sudah terpasang.Biarkan komputer booting, kemudian pastikan masuk ke dalam menu installer.Lakukan pemilihan partisi pada hardisk, apabila belum terdapat partisi, maka buat partisi baru.Pilih partisi, tunggu hingga proses installasi selesaiLakukan pengaturan tambahan mulai dari pemberian nama komputer atau laptop, pemberian password.Selesai

Pada saat proses installasi Windows 10, komputer akan membutuhkan restart sebanyak 1 hingga 2 kali restart. Untuk melakukan beberapa pengaturan pada komputer setelah proses installasi selesai, pengaturan tambahan pada Windows 10 dapat dilakukan melalui menu Control Panel.

Pembahasan

Sistem operasi windows 10 adalah salah satu sistem operasi yang banyak digunakan, selain sistem operasi windows 10. Microsoft juga memiliki sistem operasi lainnya, yaitu

Windows XPWindows VistaWindows 7Windows 8

Supaya proses installasi berjalan dengan lancar, komputer harus memiliki hardware dengannspesifikasi minimal, berikut ini adalah syarat spesifikasi minimal komputer untuk bisa menjalankan Windows 10.

Processor dengan kecepatan 1 GHzRAM  1 GB untuk 32 bit dan 2 GB untuk 64 bit.Kapasitas hardisk sebesar 16 GB untuk 32 bit dan 20 GB untuk 64 bit.
