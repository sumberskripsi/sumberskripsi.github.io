Jawab:

Penjelasan dengan langkah-langkah:

Luas permukaan kerucut :

Phi x r x (s + r)

s = garis pelukis

r = jari - jari

Maka :

22/7 x 14/2 x (10 + 14/2)

22/7 x 7 x (10 + 7)

22 x 17

374 cm^2.

Jadi, luas permukaan kerucut tersebut adalah 374 cm^2.
