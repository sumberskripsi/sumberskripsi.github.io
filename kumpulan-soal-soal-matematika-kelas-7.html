15. nilai rata-rata ujian matematika di kelas A adalah 7 seseorang mengikuti ujian susulan dan mendapatkan nilai 8 jika rata rata di kelas A menjadi 7.05 maka banyak siswa dalam kelas A adalah .... anak kak tolong di jawab ya !soal nya dikumpulkan besok !


berarti :
(7n + 8(1) / (n+1) = 7,05
7n + 8 = 7,05n + 7,05
7,05n - 7n = 8 - 7,05
0,05n = 0,95
n = 0,95/0,05
n = 95/5
n = 19
banyak siswa awal = 19 orang, banyak siswa akhir = 20 orang karena nambah 1 orang ngikut

16. nilai rata rata ujian matematika di kelas A adalah 7 seseorang mengikuti ujian susulan dan mendapatkan nilai 8 jika rata rata dikelas A menjadi 7,05 maka banyak siswa dalam kelas A adalah....Anaka.12b.15c.18d.20Tolong pake cara soalnya dikumpulkan besok !PLiS


Jawab:

n = total nilai

x = jumlah anak


1. n ÷ x = 7

2. (n + 8) ÷ (x+1) = 7.05


Persamaan 1 diubah menjadi

n = 7x


Persamaan 2 diubah menjadi

n = 7.05(x+1) - 8

n = 7.05x + 7.05 - 8

n = 7.05x - 0.95


7x = 7.05x -0.95

0.5x = 0.95

x = 19


Total anak = 19 + 1 (susulan) = 20

Untuk mencari jangkauan (range) dari data tersebut, kita perlu mencari selisih antara nilai terbesar dan terkecil dalam kumpulan data.

Data nilai Matematika: 4, 8, 7, 6, 6, 7, 5, 7, 9, 8, 8, 3, 7, 3, 4, 4.

Terbesar: 9

Terkecil: 3

Jangkauan (range) = Terbesar - Terkecil

= 9 - 3

= 6

Jadi, jangkauan (range) dari data nilai Matematika tersebut adalah 6.

Selanjutnya, untuk mencari Q₁, Q₂, dan Q₃ (kuartil), kita perlu mengurutkan data tersebut terlebih dahulu.

Data terurut: 3, 3, 4, 4, 4, 5, 6, 6, 7, 7, 7, 7, 8, 8, 8, 9

Jumlah data (n) = 16

Langkah pertama, mencari Q₁ (kuartil pertama):

Q₁ = (n + 1) / 4

= (16 + 1) / 4

= 4.25

Karena 4.25 bukan bilangan bulat, kita perlu mencari nilai tengah antara urutan ke-4 dan ke-5 dalam data terurut.

Q₁ = (4 + 5) / 2

= 4.5

Jadi, Q₁ (kuartil pertama) adalah 4.5.

Langkah berikutnya, mencari Q₂ (kuartil kedua atau median):

Q₂ = (n + 1) / 2

= (16 + 1) / 2

= 8.5

Karena 8.5 bukan bilangan bulat, kita perlu mencari nilai tengah antara urutan ke-8 dan ke-9 dalam data terurut.

Q₂ = (8 + 9) / 2

= 8.5

Jadi, Q₂ (kuartil kedua atau median) adalah 8.5.

Terakhir, mencari Q₃ (kuartil ketiga):

Q₃ = (3 * (n + 1)) / 4

= (3 * (16 + 1)) / 4

= 12.75

Karena 12.75 bukan bilangan bulat, kita perlu mencari nilai tengah antara urutan ke-12 dan ke-13 dalam data terurut.

Q₃ = (12 + 13) / 2

= 12.5

Jadi, Q₃ (kuartil ketiga) adalah 12.5.

Ringkasan:
