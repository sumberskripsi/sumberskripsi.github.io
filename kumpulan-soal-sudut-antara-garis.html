Pencerminan Segitiga ABC yang Titik Sudutnya di A(3, 2), B(4, 4), dan C(1, 3)

Pencerminan atau refleksi adalah salah satu jenis transformasi geometri. Transformasi jenis ini akan memetakan suatu titik pada bangun geometri dengan menggunaan sifat benda dan bayangan pada cermin datar.

Bayangan hasil pencerminan akan memiliki bentuk dan ukuran yang sama dengan bentuk dan ukuran awalnya.

Beberapa rumus praktis untuk pencerminan:

Pencerminan terhadap sumbu X: P (x, y) → P' (x, -y)Pencerminan terhadap sumbu Y: P (x, y) → P' (-x, y)Pencerminan terhadap titik (0, 0) : P (x, y) → P' (- x, - y)Pencerminan terhadap titik (a, b) : P (x, y) → P' (2a - x, 2b - y)Pencerminan terhadap garis x = y: P (x, y) → P' (y, x)Pencerminan terhadap garis x = -y atau y = -x: P (x, y) → P' (-y, -x)Pencerminan terhadap garis x = a: P (x, y) → P' (2a - x, y)Pencerminan terhadap garis y = a: P (x, y) → P' (x, 2a - y)

Untuk transformasi bentuk geometri, kita cukup mentransformasikan titik-titik signifikan, seperti titik sudut, titik puncak dan lain-lain.

jadi untuk menjawab soal ini, kita cukup mencari bayangan dari titik A, titik B, dan titik C seperti di bawah ini :)


Pembahasan

Titik Sudut Segitiga ABC adalah: A (3, 2), B (4, 4), dan C (1, 3)


Pencerminan Terhadap Sumbu X

Gunakan rumus 1: P (x, y) → P' (x, -y)

Nilai x tetap, nilai y berubah tanda.

Sehingga:

Titik A (3, 2)  → A' (3, -2)Titik B (4, 4)  → B' (4, -4)Titik C (1, 3)  → C' (1, -3)

Pencerminan Terhadap Sumbu Y

Gunakan rumus 2: P (x, y) → P' (-x, y)

Nilai x berubah tanda, nilai y tetap.

Sehingga:

Titik A (3, 2)  → A' (-3, 2)Titik B (4, 4)  → B' (-4, 4)Titik C (1, 3)  → C' (-1, 3)

Pencerminan Terhadap Titik Asal O (0, 0)

Gunakan rumus 3: P (x, y) → P' (- x, - y)

Nilai x berubah tanda, nilai y berubah tanda.

Sehingga:

Titik A (3, 2)  → A' (-3, -2)Titik B (4, 4)  → B' (-4, -4)Titik C (1, 3)  → C' (-1, -3)

Pencerminan Terhadap Garis y = x

Gunakan rumus 5: P (x, y) → P' (y, x)

Nilai x' adalah nilai y, nilai y' adalah nilai x.

Sehingga:

Titik A (3, 2)  → A' (2, 3)Titik B (4, 4)  → B' (4, 4)Titik C (1, 3)  → C' (3, 1)

Pencerminan Terhadap Garis y = -x

Gunakan rumus 6: P (x, y) → P' (-y, -x)

Nilai x' adalah nilai -y, nilai y' adalah nilai -x.

Sehingga:

Titik A (3, 2)  → A' (-2, -3)Titik B (4, 4)  → B' (-4, -4)Titik C (1, 3)  → C' (-3, -1)

Pencerminan Terhadap Garis Y = 2

Gunakan rumus 8: P (x, y) → P' (x, 2a - y). Karena a = 2, maka P' (x, 2.2 - y) atau P' (x, 4 - y)

Nilai x' tetap, nilai y' = 4 -y.

Sehingga:

Titik A (3, 2)  → A' (3, 4 - 2) atau A' (3, 2)Titik B (4, 4)  → B' (4, 4 - 4) atau B' (4, 0)Titik C (1, 3)  → C' (1, 4 - 3) atau C' (1, 1)

Pencerminan Terhadap Garis X = 3

Gunakan rumus 8: P (x, y) → P' (2a - x, y). Karena a = 3, maka P’ (3.2 – x, y) atau P’ (6 – x, y)

Nilai x' = 6 - x, nilai y' tetap.

Sehingga:

Titik A (3, 2)  → A' (6 - 3, 2) atau A' (3, 2)Titik B (4, 4)  → B' (6 - 4, 4) atau B' (2, 4)Titik C (1, 3)  → C' (6 - 1, 3) atau C' (5, 3)
