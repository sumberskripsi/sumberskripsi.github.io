52. BaCl2 + Na2SO4 -> BaSO4 + 2NaCl

untuk mengetahui ini reaksi redoks atau bukan maka kita mencari biloks (bilangan oksidasi) ya dulu

BaCl2 = +2Na2SO4 = +4BaSO4 = +82NaCl = +1

Nah setelah menentukan biloks kita tentukan reduksi dan oksidasi yang mana.

BaCl2 --> BaSO4 mengalami kenaikan biloks yg dari +2 ke +8 maka ini disebut oksidasi.

Na2SO4 --> 2NaCl mengalami penurunan biloks yg dari +4 ke +1 maka ini disebut reduksi.

Jadi, persamaan yang merupakan biloks adalah yang C

2. SOAL UMPN KIMIA POLBAN (UMPN 2019)


Jawaban:

122,5gram

Penjelasan:

pertama buat mula2, reaksi/terurai, sisa

kemudian cari mol KClO3 dan ubah ke gram

Jawaban:

(1) B. 122,5 gram

(2) B. +11,9 kkal

Penjelasan:

(1) [tex]\boxed{\star\ Persamaan\ Reaksi\ \star}[/tex]

[tex]KClO_{3}_{(aq)}+6\ HCl_{(aq)} \rightarrow KCl_{(aq)}+3\ H_{2}O_{(l)}+3\ Cl_{2}_{(g)}[/tex]

◈ Diketahui massa KCl yang terbentuk adalah sebanyak 74,5 gram.

◈ Massa molekul relatif KCl = 39 + 35,5 = 74,5 gram/mol

◈ Mol KCl = [tex]\frac{m}{Mr}[/tex] = [tex]\frac{74,5\ gram}{74,5\ gram/mol}[/tex] = 1 mol

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

[tex]\boxed{\boxed{Untuk\ mencari\ mol\ KClO_{3}, gunakan\ perbandingan\ koefisien\ pada\ reaksi.}}[/tex]

Mol KClO₃ = [tex]\frac{Koefisien\ KClO_3}{Koefisien\ KCl}\times mol\ KCl[/tex]

Mol KClO₃ = [tex]\frac{1}{1}\times 1\ mol[/tex]

Mol KClO₃ = [tex]1\ mol[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Mencari massa KClO₃ yang digunakan: (Mr KClO₃ = 122,5 gram/mol)

Mol KClO₃ = 1 mol

Mol KClO₃ = [tex]\frac{m}{Mr}[/tex]

Massa KClO₃ = [tex]mol \times Mr[/tex]

Massa KClO₃ = [tex]1\ mol \times 122,5\ gram/mol[/tex]

[tex]\boxed{Massa\ KClO_3 = 122,5\ gram}[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Jadi, massa kalium klorat yang diperlukan untuk menghasilkan 74,5 gram kalium klorida sebanyak dalam reaksi ini adalah sebanyak 122,5 gram.

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

(2) Diketahui perubahan entalpi untuk reaksi:

[tex]H_2O_{(l)} \rightarrow H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\ \ \Delta H = +68,3\ kkal\\H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\rightarrow H_2O_{(g)}\ \ \Delta H = -57,8\ kkal\\H_2O_{(l)} \rightarrow H_2O_{(s)}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \Delta H = -1,4\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Perubahan entalpi dari es menjadi uap air:

[tex]H_2O_{(l)} \rightarrow H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\ \ \Delta H = +68,3\ kkal\\H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\rightarrow H_2O_{(g)}\ \ \Delta H = -57,8\ kkal\\H_2O_{(s)} \rightarrow H_2O_{(l)}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \Delta H = +1,4\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Sehingga,

[tex]H_2O_{(s)} \rightarrow H_2O_{(g)} \ \ \ \ \ \ \ \ \Delta H =(68,3-57,8+1,4)kkal[/tex]

[tex]H_2O_{(s)} \rightarrow H_2O_{(g)} \ \ \ \ \ \ \ \ \Delta H = +11,9\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Jadi, perubahan entalpi yang terjadi untuk mengubah es menjadi uap air adalah sebesar + 11,9 kkal.

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

18. 15 nomor contoh soal sudut elevasi dan depresi(soal cerita)beserta pembahsannya


15 nomor contoh soal sudut elevasi dan depresi (soal cerita) beserta pembahasannya. Sudut elevasi adalah sudut yang dibentuk jika kita melihat ke atas terhadap garis mendatar (horizontal). Sudut deperesi adalah sudut yang dibentuk jika kita melihat ke bawah terhadap garis mendatar (horizontal).  

Pembahasan  

1. Budi melihat puncak menara dengan sudut elevasi 30°. Jika jarak antara Budi dan menara yang dilihatnya adalah 150 m dan tinggi Budi adalah 120 cm maka tinggi menara tersebut adalah …

Jawab

tan 30⁰ = [tex]\frac{x}{150}[/tex]

[tex]\frac{1}{3} \sqrt{3} = \frac{x}{150}[/tex]

x = [tex]\frac{1}{3} \sqrt{3}[/tex]  . 150  

x = 50√3  

Jadi tinggi menara adalah

= x + tinggi Budi

= 50√3 m + 120 cm

= 50√3 m + 1,2 m

= (50√3 + 1,2) m

2. Andi berdiri tegak pada jarak 10√3 m dari kaki sebuah pohon besar yang tumbuh gerak lurus. Jika tinggi Andi 1,6 m dan melihat ke puncak pohon dengan sudut elevasi 60°. Tentukan tinggi pohon tersebut?  

Jawab

tan 60⁰ = [tex]\frac{x}{10\sqrt{3}}[/tex]

[tex]\sqrt{3} = \frac{x}{10\sqrt{3}}[/tex]

x = √3 . 10√3

x = 30

Jadi tinggi pohon adalah

= x + tinggi Andi

= 30 m + 1,6 m

= 31,6 m

3. Sebuah gedung yang tingginya 50 m dan terdapat sebuah batu besar di dekat gedung. Jika sudut depresi dari titik puncak gedung terhadap batu tersebut adalah 30⁰ maka jarak batu terhadap dasar gedung tersebut adalah …

Jawab

tan 30⁰ = [tex]\frac{50}{x}[/tex]

[tex]\frac{1}{\sqrt{3}} = \frac{50}{x}[/tex]

x = 50√3

Jadi jarak batu terhadap dasar gedung tersebut adalah 50√3 m

4. Iwan memandang puncak sebuah gedung dengan sudut elevasi 60⁰. Tinggi orang Iwan 1,7 m dan jarak Iwan dengan gedung 40√3 m. Tinggi gedung adalah ….

Jawab

tan 60⁰ = [tex]\frac{x}{40\sqrt{3}}[/tex]

[tex]\sqrt{3} = \frac{x}{40\sqrt{3}}[/tex]

x = √3 . 40√3

x = 120  

Jadi tinggi gedung adalah

= x + tinggi Iwan

= 120 m + 1,7 m

= 121,7 m

5. Seorang siswa diberikan tugas untuk mengukur tinggi sebuah gedung dengan menggunakan klinometer pada awal berdiri melihat ujung atas gedung dengan sudut elevasi  30° kemudian mendekati gedung sejauh 20 m dengan sudut elevasi  45°, jika tinggi siswa tersebut 1,5 m maka tinggi gedung adalah …

Jawab

Untuk sudut 45°

tan 45⁰ = [tex]\frac{x}{y}[/tex]

1 = [tex]\frac{x}{y}[/tex]

y = x

Untuk sudut 30°

tan 30⁰ = [tex]\frac{x}{20 + y}[/tex]

[tex]\frac{1}{\sqrt{3}}= \frac{x}{20 + x}[/tex]

√3 x = 20 + x

√3 x – x = 20

x(√3 – 1) = 20

x = [tex]\frac{20}{\sqrt{3} - 1}[/tex]

x = [tex]\frac{20}{\sqrt{3} - 1} \times \frac{\sqrt{3} + 1}{\sqrt{3} + 1} [/tex]

x = [tex]\frac{20(\sqrt{3} + 1)}{3 - 1} [/tex]

x = [tex]\frac{20(\sqrt{3} + 1)}{2} [/tex]

x = 10(√3 + 1)

x = 10√3 + 10

Jadi tinggi gedung tersebut adalah

= x + tinggi siswa

= (10√3 + 10 + 1,5) m

= (10√3 + 11,5) m

6. Seorang anak dengan tinggi 160 cm berdiri pada jarak 12 m dari kaki tiang bendera. Jika sudut depresi dari puncak tiang terhadap anak adalah 45° maka tinggi tiang bendera itu adalah …

1 = [tex]\frac{x}{12}[/tex]

Penjelasan:

Pertama, cari jumlah mol masing-masing zat yang bereaksi.

Mol Cu = [tex]\frac{m}{Ar}[/tex] = [tex]\frac{40\ gram}{64\ sma}[/tex] = 0,625 mol

Mol S = [tex]\frac{m}{Ar}[/tex] = [tex]\frac{4\ gram}{32\ sma}[/tex] = 0,125 mol

⟡ ⟡ ⟡

Kemudian, kita reaksikan kedua zat tersebut.

[tex]\boxed{\bold{Persamaan\ Reaksi}}[/tex]

[tex]\ \ \ \ \ \ 2Cu_{(s)}\ +\ \ \ S_{(s)}\ \ \ \rightarrow\ \ Cu_2S_{(s)}\\m\ \ .625\ mol\ \ \ .125\ mol\ \ \ \ \ \ \ \ \ - \\r\ \ \ .250\ mol\ \ \ .125\ mol\ \ \ \ \ \ .125\ mol\\s\ \ \ .375\ mol\ \ \ \ \ \ \ -\ \ \ \ \ \ \ \ \ \ .125\ mol[/tex]

⟡ ⟡ ⟡

Di akhir reaksi masih terdapat sisa mol Cu sebanyak 0,375 mol. Sehingga massa zat yang tidak bereaksi tersebut adalah:

Massa Cu = mol Cu sisa × Ar

Massa Cu = 0,375 mol × 64 sma

[tex]\boxed{Massa\ Cu=24\ gram}[/tex]

⟡ ⟡ ⟡
