32. tipe file yang dapat disertakan sebagai attachment adalah..


Kelas: IX
Mata pelajaran: Teknologi Informasi
Materi:Â  Email

Saya akan mencoba menjawab dengan dua jawaban:

Jawaban pendek:

Jenis atau tipe file yang dapat disertakan sebagai attachment adalah file text biasa (.txt), dokumen (.doc, .docx, .pdf), file gambar (.jpg, .bmp, .gif, .png), spreadsheet (.xls, .xlsx), presentasi (.ppt, .pptx), kompresi (.zip, .rar) dan file lain kecuali file jenis tertentu yang diblok seperti file aplikasi (.exe).

Jawaban panjang:

Attachment adalah file yang dapat dilampirkan ketika kita mengirimkan email. Ketika mengirimkan email kita bisa melampirak satu atau beberapa file ke tujuan yang nantinya fapat diunduh oleh tujuan.

Apakah suatu file bisa dikirimkan atau tidak tergantung pada ukuran jan jenis file. Untuk ukuran, file-file yang dilampirkan tidak boleh melebihi batas ukuran yang ditetapkan penyedia jasa email baik di sisi pengirin maupun penerima.

Demikian juga dengan jenis atau tipe file yang dilampirkan, harus diperbolehkan untuk dilampirkan oleh penyedia jasa email baik di sisi pengirin maupun penerima.

Umumnya file yang dapat dilampirkan memiliki jenis atau tipe:

1. text biasa (.txt),
2. dokumen (.doc, .docx, .pdf),
3. file gambar (.jpg, .bmp, .gif, .png),
4. Spreadsheet (.xls, .xlsx),
5. presentasi (.ppt, .pptx),
6. kompresi (.zip, .rar)
7. file lainya.

Penyedia jasa email seperti Gmail dan Yahoo tidak mengijinkan file aplikasi executable dengan ektensi .exe. Ini karena file aplikasi yang dikirim lewat email sering mengandung virus, spyware dan program berbahaya lainya.

Beberapa penyedia jasa email seperti Gmail dan Outlook dapat mendukung preview atau pratinjau dari file attachmen dalam format tertentu seperti dokumen atau gambar.

Selain file attachment dapat berupa link ke situs lain atau link ke dokumen yang disimpan di penyedia jasa penyimpanan seperti Google Drive, Dropbox atau Microsoft OneSpace.
