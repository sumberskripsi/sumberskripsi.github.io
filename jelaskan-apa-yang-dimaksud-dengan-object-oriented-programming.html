Isi / Body method berbeda.

Biasanya berbeda class. interface dan implement.

First Class

Class OverRiding {

public OverRiding(){}

/*

* First Method

*/

public void firstMethod ( ) {

System.out.println("Method on class OverRiding is calling ");

}

}

Second Class

Class SubOverRiding extends OverRiding {

/*

* Override First Method on class OverRiding

*/

public void firstMethod () {

System.out.println("This method override ");

System.out.println("Method on class SubOverRiding in calling ");

}

/*

* Main Method

*/

Public static void main (String [ ] args) {

new OverRiding().firstMethod();

new SubOverRiding().firstMethod();;

Yang dimaksud dengan profit oriented, social oriented dan komersial adalah

Profit oriented adalah tujuan perusahaan untuk mencetak laba yang sebesar-besarnya.Social oriented adalah tujuan perusahaan untuk kehidupan sosial masyarakat, atau bertujuan untuk amal.Komersial oriented adalah tujuan perusahaan yang berkaitan dengan menghasilkan keuangan yang lebih, hampir mirip dengan profit oriented.Pembahasan

Perusahaan didirikan tentu mempunyai berbagai tujuan, ada yang bertujuan pada keuntungan, ada yang bertujuan pada kehidupan sosial masyarakat dan amal, dan ada juga bertujuan secara komersial, atau menghasilkan uang yang dapat digunakan untuk kekayaan pribadi dan juga tujuan kemanusiaan.

Masing-masing tujuan tersebut dibuat sesuai dengan visi dan misi masing-masig perusahaan, misalnyasebuah Non Profit Organiztion, tentunya didirikan dengan tujuan untuk beramal, dimana mereka mencari donatur kemudian uang yang didapat dari donatur tersebut digunakan untuk tujuan sosial, misalnya mengentaskan kemiskinan, membuat sekolah bagi masyarakat tidak mampu dan lain sebagainya.

Begitu juga perusahaan yang bertujuan untuk mendapatkan laba, maka mereka berusaha dan bekerja keras membangun perusahaan tersebut, mengembangkan bisnisnya untuk mencapai laba yang sebesar besarnya.
