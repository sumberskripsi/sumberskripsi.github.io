10. intan mengikuti Olimpiade Matematika tingkat SD di kotanya ​


Penjelasan dengan langkah-langkah:

34 soal benar

11 soal salah

maka tersisa 5 soal yang tidak dijawab intan

maka nilai intan

34 (2) + 11 (-1) + 5 (0) =

68 - 11 + 0 =

nilai intan = 57

Jawaban:

• Diketahui:

Jawaban benar = skor 2

Jawaban salah = skor -1

Tidak dijawab = skor 0

Soal benar = 34 soal

Soal salah = 11 soal

Jumlah soal = 50

• Ditanya:

Skor yang dimiliki Intan?

• Jawab:

« Jawaban yang tidak dijawab

Jumlah soal = Jawaban benar + jawaban salah – jawaban yang tidak dijawab

=> 50 = 34 + 11 – x

50 = 45 – x

x = 50 – 45

x = 5

« Intan berhasil menjawab

Jawaban benar = 34 × 2 = 68

Jawaban salah = 11 × (-1) = -11

Jawaban yang tidak dijawab = 5 × 0 = 0

« Jumlah skor

= Jawaban benar + jawaban salah + jawaban yang tidak dijawab

= 68 + (-11) + 0

= 57 skor

Jadi, skor yang dimiliki Intan adalah 57 skor

r = 3

nPr = n!

(n – r)!

40P3 = 40!

(40 – 3)!

40P3 = 40!

37!

37!

40P3 = 59.280 ✔

Banyak susunan.

= 40!/37!

= (40×39×38×37!)/37!

= 1.560×38

= 59.280 susunan
