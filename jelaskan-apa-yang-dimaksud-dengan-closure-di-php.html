4. if…else

Merupakan statement kondisi yang digunakan untuk menentukan aksi yang akan dilakukan pada kondisi tertentu. PHP memiliki 4 jenis statemen kondisi, yaitu :

If – digunakan untuk menjalankan kode-kode berikutnya hanya jika kondisi bernilai benar (true).

Syntax dasar

If (condition) kode yang akan dijalankan ketika kondisi bernilai benar;

Contoh:

<?php

$t=date("T");

If ($t=="Sabtu") ;

Echo "Selamat Liburan broo..";

?>

If…else – digunakan untuk menjalankan kode-kode berikutnya jika kondisi bernilai benar (true) dan akan menjalankan kode-kode yang lain jika kondisi bernilai salah (false).

Syntax dasar:

If (condition) kode yang akan dijalankan ketika kondisi bernilai benar;

Else

Kode yang akan dijalankan ketika kondisi bernilai salah;

Contoh:

<?php

$t=date("T");

If ($t=="Sabtu")

Echo "Selamat Liburan broo..";

Else

Echo "Hari yang cerah sekali";

?>

If..else – digunakan untuk memilih salah satu dari beberapa blok kode yang akan dijalankan.

Syntax dasar:

If (condition) kode yang akan dijalankan ketika kondisi bernilai benar;

Elseif (condition) kode yang akan dijalankan ketika kondisi berikutnya bernilai benar;

Else

Kode yang akan dijalankan ketika kondisi benilai slaah;

Contoh:

<?php

$t=date("T");

If ($t=="Sabtu");

  Else "Selamat liburan coyy..";

Elseif ($t=="Senin");

Echo "Senin Ceria";

Else

Echo "Hari yang menyenangkan";

?>


14. Apa yang dimaksud dengan variabel dalam PHP?


Dalam pemograman, variabel adalah suatu lokasi penyimpanan (di dalam memori komputer) yang berisikan nilai atau informasi yang nilainya tidak diketahui maupun telah diketahui . Dalam defenisi bebasnya, variabel adalah kode program yang digunakan untuk menampung nilai tertentu. Nilai yang disimpan di dalam variabel selanjutnya dapat dipindahkan ke dalam database, atau ditampilkan kembali ke pengguna. Nilai dari variabel dapat di isi dengan informasi yang diinginkan dan dapat dirubah nilainya pada saat kode program sedang berjalan. Sebuah variabel memiliki nama yang digunakan untuk mengakses nilai dari variabel itu. Jika anda memiliki pengetahuan dasar tentang bahasa pemograman, tentunya tidak asing dengan istilah variabel. Sama seperti variabel dalam bahasa pemograman lainnya, variabel dalam PHP digunakan untuk menampung nilai inputan dari user, atau nilai yang kita definisikan sendiri. Namun PHP memiliki beberapa aturan tentang cara penggunaan dan penulisan variabel.
Variable sering dijumpai dalam bahasa pemograman variable ini digunakan untuk menyimpan data yang bersifat sementara baik jenis string, integer, array dll. Variable dalam script PHP dinyatakan dengan tanda $ didepan nama variable.

29. apa perbedaan php mysql dengan php mysqli ?


Perbedaan Antara mysql extension Dengan mysqli extension

Sepanjang tutorial PHP MySQL Part 4 sampai 11 di duniailkom ini, kita menggunakan extension mysql dari PHP untuk mengakses MySQL. Fungsi-fungsi yang kita pelajari tersebut (seperti fungsi mysql_connect, mysql_query, dan mysql_fetch_array) memang sudah tidak disarankan lagi, tetapi saya tetap membahasnya sebagai dasar bagi kita untuk masuk ke extension yang lebih baru: mysqli dan PDO.

Mysqli merupakan kependekan dari MySQL Improved Extension. Seperti yang terlihat dari namanya, extension ini merupakan versi perbaikan dan penambahan dari extension mysql sebelumnya yang umum digunakan. Extension mysqli dibuat untuk mendukung fitur-fitur terbaru dari MySQL Server versi 4.1 keatas.

37. komponen apasajakah yang diperlukan dalam menjalankan php? sebutkan dan jelaskan​


Secara umum, untuk dapat menjalankan skrip PHP hanya membutuhkan hal berikut

WebServer (Seperti: Apache, Nginx, IIS, dll)

PHP (pastinya)

Meskipun tak wajib, kemungkinan besar kamu juga akan membutuhkan sebuah Database seperti MariaDB atau MySQL sebagai fasilitas penyimpanan data.

Setelah semua tersedia, maka PHP akan melakukan tugasnya yaitu melakukan eksekusi terhadap semua file dengan tipe .php yang telah disimpan dalam direktori tertentu.  

Baiklah sobat webmaster, semoga artikel yang singkat ini dapat bermanfaat untuk kita bersama. Pastikan kamu banyak belajar tentang webserver dan memastikan pula PHP telah tertanam didalamnya, karena jika PHP tidak dijalankan oleh webserver maka file .php tidak akan dieksekusi, biasanya justru malah menampilkan sintak PHP yang masih dalam bentuk script. Kita akan belajar lebih dalam tentang webserver dilain kesempatan.

Singkatnya LAMP (Linux, Apache, Mysql, PHP) , atau WAMP (Windows, Apache, Mysql, PHP). merupakan salah satu paket untuk memenuhi kebutuhan webserver.

Linux/Windows : Sebagai operating system yang mengendalikan segala aspek dari komputer

Apache: Berfungsi sebagai HTTP Server, yang bertanggung jawab akan request-response terhadap permintaan client-server dan sebaliknya.
