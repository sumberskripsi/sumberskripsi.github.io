2. matematika hal 93 kelas 8 kurikulum 2013


Lengkapi tabel berikut halaman 93 Buku Matematika kelas 8 smp semester 2

Pada bab ini kita masih membahas soal Lingkaran, yaitu tentang sudut pusat, panjang busur dan Luas Juring

Sudut pusat adalah sudut yang dibentuk oleh dua jari-jari yang berpotongan pada pusat lingkaran

Busur adalah garis lengkung yang menghubungkan dua titik sembarang pada lengkungan lingkaran

Juring adalah daerah di dalam lingkaran yang dibatasi oleh dua jari-jari lingkaran dan busur lingkaran dihadapan sudut pusat yang dibentuk oleh kedua jari-jarinya

Rumus :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

Busur = [tex]\frac{\theta}{360}[/tex]K

Dimana :

[tex]\theta[/tex] = sudut juring dalam derajat

K = keliling lingkaran (K = 2πr )

L = Luas lingkaran (L = πr² )

Pembahasan :

1. a. Diketahui :

[tex]\theta[/tex] = 90°

r = 7cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 7

= 44 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{90}{360}[/tex] x 44

= 11 cm


b. Diketahui :

[tex]\theta[/tex] = 60°

r = 21cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 21

= 132 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{60}{360}[/tex] x 132

= 22 cm


c. Diketahui :

[tex]\theta[/tex] = 120°

Panjang Busur = 88 cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

88 = [tex]\frac{120}{360}[/tex] x K

88 = [tex]\frac{1}{3}[/tex] x K

K = 264 cm

Keliling = 2πr

264 = 2 x [tex]\frac{22}{7}[/tex] x r

264 = [tex]\frac{44}{7}[/tex] x r

r = 42 cm


d. Diketahui :

r = 100 cm

Panjang Busur = 31,4 cm

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

Keliling = 2πr

= 2 x 3,14 x 100

= 628 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

314 = [tex]\frac{\theta}{360}[/tex] x 628

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{314}{628}[/tex]

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{1}{2}[/tex]

[tex]\theta[/tex] = [tex]\frac{360}{2}[/tex]

[tex]\theta[/tex] = 180°


e. Diketahui :

[tex]\theta[/tex] = 72°

Panjang Busur = 1.256 cm

π = 3,14

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

1.256 = [tex]\frac{72}{360}[/tex] x K

1.256 = [tex]\frac{1}{5}[/tex] x K

K = 6.280 cm

Keliling = 2πr

6.280 = 2 x 3,14 x r

6.280 = 6,28r

r = 1000 cm


2. a. Diketahui :

[tex]\theta[/tex] = 100°

r = 6cm

π = 3,14

Ditanya : Luas Juring ?

Dijawab :

L = πr²

L = 3,14 x 6 x 6

L = 113,04 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

= [tex]\frac{100}{360}[/tex] x 113,04

= 31,4 cm²


b. Diketahui :

[tex]\theta[/tex] = 25°

Luas juring = 31,4 cm²

π = 3,14

Ditanya : r ?

Dijawab :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

31,4 = [tex]\frac{25}{360}[/tex] x L

31,4 = [tex]\frac{5}{72}[/tex] x L

L = 452,16 cm²

L = πr²

452,16 = 3,14 x r²

r² = 144

r = 12 cm


c. Diketahui :

r = 90 cm

Luas juring = 8.478 cm²

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

L = πr²

L = 3,14 x 90 x 90

L = 25.434 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

8.478 = [tex]\frac{\theta}{360}[/tex] x 25.434

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{8.478}{25.434}[/tex]

[tex]\theta[/tex] = 120°

13. PP, bantu dong soal ulangan kelas 8 kurikulum 2013 ​


Jawaban soal 1:

11, 15, 19, 23, 27, 31, 35 memiliki rumus suku ke-n yaitu Un=4n + 7 dan untuk suku ke-22 hasilnya adalah 95, sementara jumlah 13 suku pertamanya adalah 455.

Jawaban soal 2:

3, 8, 13, 18 memiliki rumus suku ke-n yaitu Un=5n - 2 dan untuk suku ke-8 dan ke-30 adalah 38 dan 148. Nilai n ketika Un=28 adalah 6. Untuk jumlah 10 suku pertamanya adalah 255.

 

Penjelasan dengan langkah-langkah:

Rumus suku ke-n adalah Un = a + (n-1)b

Rumus jumlah suku ke-n adalah Sn = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

Keterangan:

a = suku pertama atau U1

b = beda atau selisih

Diketahui:

Soal 1: 11, 15, 19, 23, 27, 31, 35

Soal 2: 3, 8, 13, 18

Ditanya:

Soal 1:

a) dan b) kosong

c) Rumus suku ke-n

d) Suku ke-22

e) Jumlah 13 suku pertama

Soal 2:

a) Tentukan garisnya hingga ke U8

b) Tentukan rumus suku ke-n

c) Jika Un= 28, tentukan nillai n

d) Tentukan suku ke-30

e) Jumlah 10 suku pertama

Jawab:

Jawaban Soal 1:

11, 15, 19, 23, 27, 31, 35

dari soal didapat informasi bahwa:

a = 11

b = U2-U1 = 15-11 = 4

maka;

c) Un = a + (n-1)b

   Un = 11 + (n-1)4

   Un = 11 + 4n - 4

   Un = 4n + 7

d) Un   = 4n + 7

   U22 = 4(22) + 7

   U22 = 88 + 7

   U22 = 95 (suku ke-22 adalah 95)

e)  Sn  = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

    S13 = [tex]\frac{13}{2}[/tex] (2x11 + ((13-1)4))

    S13 =  [tex]\frac{13}{2}[/tex] (22 + 48)

    S13 =  [tex]\frac{13}{2}[/tex] (70)

    S13 = 455 (jumlah 13 suku pertamanya adalah 455)

Jawaban Soal 2:

3, 8, 13, 18

dari soal didapat informasi bahwa:

a = 3

b = U2-U1 = 8-3 = 5

maka;

a) Untuk melengkapi baris atau garis hingga U8 maka sederhananya tinggal menambahkannya dengan beda atau selisih 5, sehingga hasilnya:

3, 8, 13, 18, 23, 28, 33, 38

b) Un = a + (n-1)b

   Un = 3 + (n-1)5

   Un = 3 + 5n - 5

   Un = 5n - 2

c) Un = a + (n-1)b

   28 = 3 + (n-1)5

   28 = 3 + 5n - 5

   5n = 30

   n   = 6

d) Un   = 5n - 2

   U30 = 5(30) - 2

   U30 = 150 - 2

   U30 = 148 (suku ke-30 adalah 148)

e)  Sn  = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

    S10 = [tex]\frac{10}{2}[/tex] (2x3 + ((10-1)5))

    S10 =  [tex]\frac{10}{2}[/tex] (6 + 45)

    S10 =  5 (51)

    S10 = 255 (jumlah 10 suku pertamanya adalah 255)

21. jawaban soal matematika kelas 7 kurikulum 2013 hal 86


Jawaban "Ayo Kita Berlatih 1.7" B. Soal Uraian, matematika kelas 7 kurikulum 2013 hal 86.

Bilangan adalah suatu konsep dalam ilmu matematika yang digunakan untuk pencacahan dan pengukuran.

Pembahasan

1.  Nyatakan bilangan berpangkat tersebut menjadi bilangan desimal

a.  10⁸ = 100.000.000  

b.  5⁸ = 5 × 5 × 5 × 5 × 5 × 5 × 5 × 5 = 390.625

c.  -10⁶ = -(10⁶) = -(1.000.000) = -1.000.000       (hasilnya negatif, karena tak dalam kurung bilangan awalnya)

d.  2⁴ × 10⁷ = 16 × 10.000.000 = 160.000.000

e.  (-2)⁴ × (-3)⁵ = 16 × (-243) = -3.888
