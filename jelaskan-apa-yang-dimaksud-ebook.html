Sebutkan dan jelaskan 15 format ebook yg didukung oleh calibre

Format buku yang didukung oleh Ebook Readr Calibre antara lain:

1. EPUB
Epub adalah salah satu format untuk aplikasi e-book yang bersifat terbuka, keistimewaan format epub dibandingkan dengan yang lain adalah kemampuan untuk melakukan optimasi tampilan text menyesuaikan diri dengan device yang dipakai untuk membuka ebook.

2. MOBI
Mobipocket e-book format berdasarkan ebook Terbuka standar menggunakan XHTML dan dapat mencakup JavaScript dan bingkai.

3. AZW
Sebuah format proprietary Amazon. Hal ini menyerupai format MOBI kadang-kadang dengan dan kadang-kadang tanpa menyertakan Digital Rights Management (DRM). DRM pada format ini dikhususkan untuk Kindle Amazon.
4. PRC
Palm Resource File. Sering menyertakan e-book Mobipocket tapi kadang-kadang menyertakan eReader atau e-book AportisDoc.
5. LRF
Format buku digital yang digunakan oleh Sony Corporation . Ini adalah format proprietary, tetapi beberapa perangkat lunak pembaca untuk-tujuan komputer umum, terutama di Linux (misalnya, internal pemirsa kaliber), memiliki kemampuan untuk membacanya. Ekstensi file lrx merupakan e-book terenkripsi DRM.

6. PDB
Software TealPoint milik pembaca untuk Palm OS. Selain format sendiri, ini akan membuka file teks biasa dan PalmDoc.

7. CBR
Berkas Comic Book Archive file atau ComicBook Reader terdiri dari serangkaian file gambar, biasanya PNG (kompresi lossless) atau JPEG (kompresi lossy) file, disimpan sebagai file arsip tunggal, untuk tujuan melihat urutan gambar, buku komik terutama.

8. CHM
Compressed HTML, sering digunakan untuk file bantuan Windows. Hal ini telah menjadi sangat populer untuk distribusi teks dan bahan pendukung lainnya melalui Web.

9. DJVU
Format djvu (dibaca DeJaVu) adalah format file komputer yang dirancang untuk menyimpan dokumen hasil scan, terutama yang berisi kombinasi text, gambar garis dan foto.

10. FB2
FictionBook adalah populer XML-based e-book format, didukung oleh pembaca gratis seperti FBReader , Haali Reader, STDU Viewer, dan Calibre.

11. HTML
Hyper Text Markup Language adalah tulang punggung dari World Wide Web. Banyak teks yang didistribusikan dalam format ini. Selain itu, beberapa pembaca e-book mendukung Cascading Style Sheets (CSS) yang pada dasarnya gayautama panduan  untuk halaman HTML.

12. LIT
Format LIT merupakan format dari Microsoft Reader yang memungkinkan teks dalam buku elektronik disesuaikan dengan lebar layar divais mobil yang digunakan untuk mebacanya.

13. PDF
Portable Document Format yang diciptakan oleh Adobe untuk produk Acrobat mereka. Format ini secara tidak langsung merupakan format yang digunakan untuk pertukaran dokumen.

14. ODT
Open Document Format atau dikenal juga sebagai Open Document atau ODF saja adalah dokumen berformat terbuka. Merupakan file berformat XML yang digunakan sebagai standar format file terbuka pada aplikasi-aplikasi Perkantoran (Office Suite).

15. TXT
.txt merupakan format standar yang dipakai untuk menyimpan seluruh teks yang belum diformat. Format ini biasanya dikenal oleh sebagian besar perangkat lunak.

16. 10 jenis format ebook yang banyak digunakan beserta penjelasannya


1. AZW - Sebuah format proprietary Amazon. Buku digital jenis ini biasanya digunakan untuk perangkat  dari amazon. Hal ini menyerupai format MOBI kadang-kadang dengan dan kadang-kadang tanpa menyertakan Digital Rights Management (DRM). DRM pada format ini dikhususkan untuk Kindle Amazon.

2. EPUB - Format terbuka didefinisikan oleh Forum Open digital book dari International Digital Publishing Forum (<idpf>). Hal ini mengacu kepada standar XHTML dan XML. Ini adalah standar yang sedang berkembang. Spesifikasi untuk EPUB dapat ditemukan di situs web IDPF. Adobe, Barnes & Noble dan Apple semua memiliki DRM mereka sendiri untuk format ini yang tidak kompatibel antara satu dengan yang lainnya. Saat ini sudah ada versi baru dari format ini disebut ePub 3 tetapi belum digunakan secara luas.

3. KF8 - Format Kindle Fire dari Amazon. Hal ini pada dasarnya ePub disusun dalam pembungkus PDB dengan Amazon DRM. Format ini diharapkan untuk digunakan bagi pembaca digital book dari Amazon lainnya.

4. MOBI – Format MobiPocket, ditampilkan menggunakan perangkat lunak membaca sendiri MobiPocket yang tersedia pada hampir semua PDA dan Smartphone. Aplikasi Mobipocket pada PC Windows dapat mengkonversi Chm, doc,Html,OCF, Pdf, Rtf, dan Txt file ke format ini. Kindle menampilkan format ini, juga.

5. PDB - Palm File Database. Dapat menyertakan beberapa format digital book yang berbeda yang ditujukan untuk perangkat berbasiskan sistem operasi Palm. Pada umumnya digunakan untuk buku digital berformat PalmDOC (AportisDoc) dan format eReader juga dan banyak lainnya.

6. PDF - Portable Document Format yang diciptakan oleh Adobe untuk produk Acrobat mereka. Ini secara tidak langsung merupakan format yang digunakan untuk pertukaran dokumen. Dukungan perangkat lunak untuk format ini hampir mencakupi semua platform komputer dan perangkat genggam. Beberapa perangkat memiliki masalah dengan PDF karena kebanyakan konten yang tersedia akan ditampilkan baik untuk format A4 atau surat, yang keduanya tidak mudah dibaca ketika diperkecil sesuai layar kecil. Beberapa aplikasi pembaca buku digital dapat menyusun ulang tampilannya beberapa dokumen PDF, termasuk Sony PRS505, untuk mengakomodasi layar kecil. Beberapa aplikasi pembaca buku digital, termasuk Iliad IREX, memiliki fitur pan-dan-zoom yang membantu mudah dibaca.

7. PRC - Palm Resource File. Sering menyertakan e-book Mobipocket tapi kadang-kadang menyertakan eReader atau e-book AportisDoc.

8. TPZ - Topaz file extension yang digunakan pada Amazon Kindle. Topaz adalah pengatur layout yang berguna untuk mengatur pada halaman, bersama dengan versi teks OCR. Format proprietary dari Amazon, digunakan untuk membuat buku-buku tua dengan cepat, karena proses konversi pada dasarnya otomatis dari scan halaman dari sebuah buku, tetapi dengan kelebihan yaitu penyusunan teks (reflows) yang sangat baik.

9. HTML - Hyper Text Markup Language adalah tulang punggung dari World Wide Web. Banyak teks yang didistribusikan dalam format ini. Selain itu, beberapa pembaca e-book mendukung Cascading Style Sheets (CSS) yang pada dasarnya gaya utama panduan untuk halaman HTML.

10. CHM - Compressed HTML, sering digunakan untuk file bantuan Windows. Hal ini telah menjadi sangat populer untuk distribusi teks dan bahan pendukung lainnya melalui Web.

11. XHTML - versi khusus dari HTML dirancang agar sesuai dengan aturan konstruksi XML. Ini adalah format standar untuk data epub.

12. XML - tujuan umum markup language untuk pertukaran data. Dalam konteks digital book umumnya terbatas pada XHTML dan RSS feed meskipun beberapa format lain yang telah ditetapkan.

21. Jelaskan cara melakukan upgrade informasi metadata ebook yang dapat anda buat dengan Calibre​


Tahapan memodifikasi informasi metadata sebuah ebook menggunakan aplikasi calibre adalah sebagai berikut

Untuk memodifikasi informasi metadata satu per satu, langkah-langkahnya adalah sebagai berikut

- Klik ebook yang akan dimodifikasi

- Klik tombol Edit Metadata atau tekan tombol E. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata

- Untuk menukar informasi judul dan pengarang secara otomatis, klik tombol diantara judul dan pengarang

- Tekan tombol disamping Author Sort untuk mengisi nilai secara otomatis. Gunakan Manage Author untuk melihat dan mengubah nilai sortir.  

- Klik tombol disamping Tag untuk menggunakan Tag Editor untuk mengatur semua tag yang terasosiasi dengan ebook

- Kotak Ids dapat digunakan untuk memasukkan ISBN (dan jenis ID lainnya).    Kotak ini akan berwarna merah jika ISBN yang dimasukkan tidak valid. Jika ISBN valid, maka kotak akan berwarna hijau

- Kotak Author Sort akan berwarna merah jika nilai sortasi berbeda dengan kaliber

Untuk memodifikasi informasi metadata semua ebook, langkah-langkahnya adalah sebagai berikut

- Pilih semua ebook yang akan dimodifikasi degan menekan tombol Ctrl atau Shift dan meng-klik ebook tersebut.  

- Klik tombol Edit Metadata. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata

- Tentukan isi dari Author, Publisher, Rating, Tags, Series, dan lain-lain. Nilai akan sama untuk semua ebook. Modifikasi ini dapat dilakukan jika semua ebook memiliki beberapa kesamaan dalam metadata

37. Jelaskan cara melakukan upgrade informasi metadata ebook yang anda buat dengan calibri?.


Untuk melakukan upgrade informasi metadata ebook yang Anda buat dengan Calibre, pertama-tama Anda perlu membuka aplikasi Calibre dan menemukan ebook yang ingin Anda upgrade metadata-nya. Kemudian, ikuti langkah-langkah berikut:

Klik kanan pada ebook yang ingin Anda upgrade metadata-nya, kemudian pilih "Edit metadata" dari menu yang muncul.Pada jendela yang muncul, Anda dapat mengubah informasi metadata ebook tersebut sesuai keinginan Anda. Dengan demikian, informasi metadata ebook yang Anda buat dengan Calibre akan ter-upgrade sesuai dengan perubahan yang Anda lakukan.

40. jelaskan tahapan memodifikasi informasi metadata sebuah ebook menggunakan aplikasi calibre?


Tahapan memodifikasi informasi metadata sebuah ebook menggunakan aplikasi calibre adalah sebagai berikut

Untuk memodifikasi informasi metadata satu per satu, langkah-langkahnya adalah sebagai berikut

Klik ebook yang akan dimodifikasi Klik tombol Edit Metadata atau tekan tombol E. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata Untuk menukar informasi judul dan pengarang secara otomatis, klik tombol diantara judul dan pengarang Tekan tombol disamping Author Sort untuk mengisi nilai secara otomatis. Gunakan Manage Author untuk melihat dan mengubah nilai sortir.   Klik tombol disamping Tag untuk menggunakan Tag Editor untuk mengatur semua tag yang terasosiasi dengan ebook Kotak Ids dapat digunakan untuk memasukkan ISBN (dan jenis ID lainnya). Kotak ini akan berwarna merah jika ISBN yang dimasukkan tidak valid. Jika ISBN valid, maka kotak akan berwarna hijau Kotak Author Sort akan berwarna merah jika nilai sortasi berbeda dengan kaliber

Untuk memodifikasi informasi metadata semua ebook, langkah-langkahnya adalah sebagai berikut

Pilih semua ebook yang akan dimodifikasi degan menekan tombol Ctrl atau Shift dan meng-klik ebook tersebut.   Klik tombol Edit Metadata. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata Tentukan isi dari Author, Publisher, Rating, Tags, Series, dan lain-lain. Nilai akan sama untuk semua ebook. Modifikasi ini dapat dilakukan jika semua ebook memiliki beberapa kesamaan dalam metadata
