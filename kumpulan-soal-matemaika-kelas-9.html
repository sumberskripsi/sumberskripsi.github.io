23. Need help matemaika Diketahui sistem persamaan linear : 1/x + 1/y = 2 2/y - 1/z = -3 1/x - 1/z = 2 nilai x+y+z adalah....


1/x = A
1/y = B
1/z = C

1/x + 1/y = 2
A + B = 2
B = 2 - A

1/x - 1/z = 2
A - C = 2
C = A - 2

2/y - 1/z = -3
2B - C = -3
2(2 - A) - (A - 2) = -3
4 - 2A - A + 2 = -3
-3A = -9
A = 3

B = 2 - A = 2 - 3 = -1
C = A - 2 = 3 - 2 = 1
