19. jelaskan sejarah ms exel ?


Sejarah
Pada tahun 1982, Microsoft membuat
sebuah program spreadsheet yang
disebut dengan Multiplan, yang sangat
populer dalam sistem-sistem CP/M,
tetapi tidak dalam sistem MS-DOS
mengingat di sana sudah berdiri
saingannya, yakni Lotus 1-2-3. Hal ini
membuat Microsoft memulai
pengembangan sebuah program
spreadsheet yang baru yang disebut
dengan Excel, dengan tujuan, seperti
yang dikatakan oleh Doug Klunder, "do
everything 1-2-3 does and do it
better/melakukan apa yang dilakukan
oleh 1-2-3 dan lebih baik lagi".

28. uraikan urutan memulai dan mengakhiri ms exel​


Jawaban:

Langkah-langkah untuk memulai Microsoft Excel 2007:

1. Mencari letak Microsoft Excel berada pada komputer, dimana lokasinya dapat berada di desktop sebagai shortcut, maupun dicari melalui Start button, All Programs, dan cari Microsoft Excel 2007.

2. Membuka dengan cara mengklik pada icon Microsoft Excel, lalu tunggu beberapa saat sampai worksheet tersedia 3. Microsoft excel siap digunakan

Langkah-langkah untuk mengakhiri Microsoft Excel 2007:

1. Saat akan mengakhiri, jika tidak ada hal yang dikerjakan dan perlu disimpan, pengguna dapat langsung keluar dengan mengklik tombol silang di pojok kanan atas window dan dapat juga dengan mengakses Office button, lalu klik tombol "Exit Excel" yang berada di paling bawah. Selain itu, pengguna juga dapat mengakhiri dengan mengklik kanan icon Excel pada taskbar, dan pilih opsi close window.

2. Jika terdapat hal yang harus disimpan atau ada perubahan, lakukan penyimpanan dengan memilih tombol Save atau Save as sesuai keperluan dimana Save untuk menyimpan perubahan dan Save as untuk menyimpan baru ataupun menyimpan sebagai suatu

nama atau jenis file lain. 3. Setelah itu melakukan penyimpanan, pengguna dapat mengakhiri dengan menekan tombol silang pada pojok kanan atas Window maupun melalui Office Button, lalu pilih tombol paling bawah yang bertuliskan "Exit Excel"

Pembahasan

Microsoft Excel merupakan salah satu perangkat lunak pengolah angka keluaran Microsoft Windows yang umum digunakan. Terdapat pula beberapa versi dalam perjalanannya hingga saat ini. Salah satu ciri khas versi 2007 ini adalah adanya Office Button disebelah kiri quick toolbar yang berisi menu untuk mempermudah pengguna dan salah satunya untuk mengakhiri program tersebut. Sedangkan, untuk memulainya, versi ini tidak jauh berbeda dimana terdapat shortcut icon yang dapat diakses di desktop, taskbar (jika di pin), maupun pada menu All programs yang dapat diakses melalui Start button pada windows
