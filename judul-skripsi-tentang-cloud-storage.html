Teknologi ini memungkinkan kita sebagai pengguna menempatkan file-file yang ada di internet. Cara ini akan memungkinkan pengguna memiliki backup file dan dapat mengaksesnya jika diperlukan.

Cloud adalah gabungan pemanfaatan teknologi komputer ('komputasi') dan pengembangan berbasis Internet ('awan'). Cloud adalah metafora dari internet, sebagaimana awan yang sering digambarkan di diagram jaringan komputer. Sebagaimana awan dalam diagram jaringan komputer tersebut, Cloud dalam Cloud Computing juga merupakan abstraksi dari infrastruktur kompleks yang disembunyikannya.

• Cloud storage membutuhkan kapasitas penyimpanan yang lebih besar, sedangkan cloud computing membutuhkan keberadaan memori dan cpu yang lebih banyak untuk melakukan pemrosesan.

• Cloud computing lebih banyak digunakan pada kebutuhan tingkat lanjut, baik bisnis atau aplikasi tertentu. Sedangkan cloud storage penggunaan penyimpanannya lebih ke data pribadi, kelompok atau pekerjaan.

• Dalam hal pengelolaannya, cloud storage lebih sederhana untuk dilakukan. Sedangkan untuk cloud computing harus memiliki atribut tambahan dalam pengelolaannya seperti akses pengguna, common line interface dan lainnya.

YouTube, Netflix, Gmail, Paypal, Spotify adalah beberapa layanan berbasis cloud yang digunakan orang setiap hari.

23. 4. Jelaskan yang dimaksud dengan cloud storage danJelaskan perbedaan dan persamaannya dengan aplikasi berbasiscloud


Cloud Storage: Cloud storage merujuk pada penyimpanan data dan file di server yang terhubung ke internet, biasanya di fasilitas data center yang dikelola oleh penyedia layanan cloud. Pengguna dapat mengunggah, menyimpan, dan mengakses data mereka melalui koneksi internet, tanpa perlu penyimpanan fisik lokal. Beberapa contoh penyedia layanan cloud storage terkenal adalah Google Drive, Dropbox, dan Amazon S3.

Perbedaan dan Persamaan dengan Aplikasi Berbasis Cloud:

Perbedaan:

1. Fungsi Utama: Cloud storage digunakan untuk menyimpan dan mengelola data, seperti file dan dokumen, sedangkan aplikasi berbasis cloud adalah perangkat lunak yang dijalankan dalam lingkungan cloud untuk menjalankan tugas-tugas tertentu, seperti pengolah kata, spreadsheet, atau komunikasi.

2. Jenis Data: Cloud storage fokus pada penyimpanan data, sementara aplikasi berbasis cloud digunakan untuk memproses data, mengolahnya, dan membuat dokumen atau informasi baru.

3. Akses Data vs. Pengolahan Data: Dalam cloud storage, Anda menyimpan data dan kemudian dapat mengaksesnya dari berbagai perangkat, tetapi tidak melakukan pengolahan data secara langsung. Dalam aplikasi berbasis cloud, data diolah untuk mencapai tujuan tertentu, seperti mengedit dokumen.

Persamaan:

1. Koneksi Internet: Kedua cloud storage dan aplikasi berbasis cloud memerlukan koneksi internet untuk mengakses dan menggunakan layanan tersebut.

2. Pengelolaan Data: Kedua jenis layanan melibatkan manajemen data, meskipun dalam konteks yang berbeda. Cloud storage memungkinkan pengguna untuk mengelola dan menyimpan data mereka, sedangkan aplikasi berbasis cloud memungkinkan pengguna untuk mengelola dan memproses data mereka.

3. Fleksibilitas dan Akses dari Berbagai Perangkat: Kedua jenis layanan ini memungkinkan akses data dan aplikasi dari berbagai perangkat, seperti komputer, tablet, atau ponsel.

27. 4. Jelaskan yang dimaksud dengan cloud storage dan Jelaskan perbedaan dan persamaannya dengan aplikasi berbasis cloud​


Cloud storage adalah layanan penyimpanan data yang disediakan oleh penyedia layanan cloud. Layanan ini memungkinkan pengguna untuk menyimpan, mengelola, dan mengakses data mereka secara online dari mana saja dan kapan saja dengan menggunakan koneksi internet.

Perbedaan antara cloud storage dan aplikasi berbasis cloud adalah pada fungsinya. Cloud storage hanya berfungsi sebagai tempat penyimpanan data, sedangkan aplikasi berbasis cloud memiliki fungsi tambahan seperti pengolahan data, pengeditan, dan berbagi data dengan pengguna lainnya.

Namun, keduanya memiliki persamaan dalam hal pengaksesannya yang sama-sama menggunakan koneksi internet dan dapat diakses dari mana saja dan kapan saja. Selain itu, keduanya juga memberikan kemudahan dalam berbagi data dengan orang lain.

37. Kelebihan menggunakan cloud storage untuk sharing file dibandingkan dengan mengirimkan melalui email adalah..


1. Kapasitas yang lebih besar: Dengan cloud storage, kamu bisa mengunggah dan menyimpan file dengan kapasitas yang jauh lebih besar dibandingkan dengan attachment file melalui email. Hal ini memungkinkan kamu untuk berbagi file yang lebih besar tanpa harus khawatir tentang pembatasan ukuran file.

2. Akses yang sama: Dengan cloud storage, semua orang yang memiliki akses ke folder atau file yang dibagikan dapat melihat dan mengeditnya secara real-time. Ini sangat berguna untuk kolaborasi dalam tim atau penggunaan bersama untuk proyek-proyek yang melibatkan banyak orang.

3. Kemudahan pengaturan: Cloud storage memungkinkan kamu untuk dengan mudah mengatur dan mengatur file dalam folder dan subfolder. Ini membuat pencarian dan pemulihan file menjadi lebih mudah dan terorganisir dibandingkan dengan mencari file tertentu di dalam email yang telah dikirim atau diterima.

4. Keamanan: Meskipun email memiliki opsi enkripsi, cloud storage sering kali memiliki tingkat keamanan dan privasi yang lebih tinggi. Kamu dapat mengatur izin akses untuk setiap file atau folder yang dibagikan, dan bekerja dengan jaringan yang aman untuk melindungi data sensitif.

5. Sinkronisasi otomatis: Cloud storage sering kali memiliki sinkronisasi otomatis dengan perangkat berbeda, sehingga kamu dapat mengakses dan mengedit file dari manapun dan kapanpun menggunakan perangkat apa pun. Ini membuatnya lebih mudah untuk berbagi dan mengakses file di perangkat seluler atau perangkat lainnya yang tidak dapat mengakses email dengan mudah.

Dalam kesimpulannya, menggunakan cloud storage untuk berbagi file memiliki kelebihan dalam kapasitas yang lebih besar, akses yang sama bagi semua pihak, kemudahan pengaturan, keamanan yang lebih baik, dan sinkronisasi otomatis dengan perangkat lainnya.
