g(-5)=7

g(2)=-7 Jadi = q(2) + r = -7    = 2q + r =-7
g(-5)=7 Jadi = q(-5) + r = 7    = -5q +r = 7

Lalu cari nilai dari q dan r
2q + r =-7  "nilai r habis"
-5q +r = 7  -

7q = -14
q = -14/7
q = -2
 
Nilai r "pakai salah satu persamaan"
2q + r =-7 "Lalu masukkan nilai q yang diperoleh tadi"
2(-2) + r = -7
-4 +r = -7
r = -7 + 4
r = -3

Rumus fungsi = g(×)=q×+r "lalu masukkan nilai q dan r"

 g(4) = -2(4) + -3
= -8 + -3

12. menyetarakan C2H6+O2(g) ---》CO2(g)+H2O(g) dengan caranya ya


Untuk menyetarakan kita mulai dari paling kiri dulu

1) [tex]C_{2}H_{2} + O_{2} ==\ \textgreater \ 2CO_{2} + H_{2}O[/tex]
    Di kiri jumlah C nya ada 2, jadi C yg di kanan di masukan koefisien nya 2

2) [tex]C_{2}H_{6} + O_{2} ==\ \textgreater \ 2CO_{2} + 3H_{2}O[/tex]
    Di kiri jumlah H nya ada 6, sedangkan di kanan H nya udah ada 2. Jadi H yg dikanan di masukan koefisien nya 3. ( 3 x 2 = 6 )

3) [tex]C_{2}H_{6} + \frac{7}{2}O_{2} ==\ \textgreater \ 2CO_{2} + 3H_{2}O[/tex]
    Jumlah O di kanan ada 7 ( 2 x 2 = 4 ) ( 3 x 1 = 3 ), sedangkan di kiri O nya baru ada 2. Jadi O yg di kiri di masukan koefisien nya 7/2 ( 7/2 x 2 = 7 )

Jadi, hasil menyetarakan nya adalah
∴ [tex]C_{2}H_{6} + \frac{7}{2}O_{2} ==\ \textgreater \ 2CO_{2} + 3H_{2}O[/tex]

27. diketahui rumus fungsi g (x) = ax + b. jika g(-4) = -13 dan g (6) = 7 maka g (-3) adalah...tolong dijawab ya dan diberi penjelasan ya ​


Jawab:

Diketahui:

fungsi g (x) = ax + bg(-4) = -13 dan g (6) = 7

Ditanya: g(-3) ?

Pembahasan

g(-4) = a(-4) + b = -13

g(6) = a(6) +b = 7

Mencari a

-4a + b = -13

6a + b = 7

----------------------- (Dikurang)

-10a = -20

a = 2

Mencari b

Misal:

-4a + b = -13

-4(2) + b = -13

-8 + b = -13

b = -5

Sehingga...

g(-3) = 2(-3) + (-5)

g(-3) = -6 - 5

g(-3) = -11

#Shiba's Solver
