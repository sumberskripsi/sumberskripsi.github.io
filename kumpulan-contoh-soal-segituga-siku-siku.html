21. ABC adalah segituga siku siku sama kaki,jika A(3,-1) B(3,5) ,maka letak titik C yg tepat adalah?​


Jawaban:

(9, -1)

Penjelasan dengan langkah-langkah:

Untuk menemukan letak titik C pada segitiga ABC yang siku-siku sama kaki, kita dapat menggunakan rumus jarak antara dua titik. Rumus tersebut adalah sebagai berikut:

d = √((x2 - x1)² + (y2 - y1)²)

Di mana:

d = jarak antara dua titik

x1, y1 = koordinat titik pertama

x2, y2 = koordinat titik kedua

Kita tahu bahwa titik A berada pada koordinat (3, -1) dan titik B berada pada koordinat (3, 5). Kita juga tahu bahwa ABC adalah segitiga siku-siku sama kaki, sehingga sisi AC sama dengan sisi AB. Dengan menggunakan rumus di atas, kita dapat menghitung jarak antara titik A dan titik B sebagai berikut:

d = √((3 - 3)² + (5 - (-1))²) = √(0 + 6²) = √36 = 6

Kita tahu bahwa sisi AC sama dengan sisi AB, yaitu 6. Kita juga tahu bahwa titik A berada pada koordinat (3, -1) dan titik B berada pada koordinat (3, 5). Oleh karena itu, kita dapat menemukan letak titik C dengan menggunakan koordinat-koordinat tersebut dan menggeser titik C 6 satuan ke kiri atau ke kanan tergantung pada keinginan kita. Jika kita menggeser titik C ke kanan maka titik C akan berada pada koordinat (9, y)

jadi letak titik C yg tepat untuk membentuk segitiga siku-siku sama kaki adalah (9, -1)
