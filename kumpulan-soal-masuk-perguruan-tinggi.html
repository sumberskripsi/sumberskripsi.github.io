Ani, Budi, dan Cica saling berbagi untuk membeli buku kumpulan soal TPA, Saintek, Soshum untuk seleksi masuk perguruan tinggi. Berdasarkan deskripsi soal sistem persamaan tiga variabel di atas, maka:

Besar uang Ani adalah Rp 131.000,00.Besar uang Budi adalah Rp 37.000,00.Besar uang Cica adalah Rp 32.000,00.

Penjelasan dengan langkah-langkah:

Diketahui:

Uang Ani Rp 30.000,00 lebih banyak dari uang Budi ditambah dua kali uang Cica.Jumlah uang Ani, Budi dan Cica adalah Rp 200.000,00.Selisih uang Budi dan Cica adalah Rp 5.000,00.

Ditanya:

Besaran uang masing-masing milik Ani, Budi dan Cica.

Dijawab:

Kita misalkan Ani = A, Budi = B, Cica = C. Kita tuliskan poin yang diketahui dalam bentuk matematis.

A - B - 2C = 30.000A + B + C = 200.000B - C = 5.000

Jadi kita memiliki sistem persamaan tiga variabel seperti di atas.

Kita dapat mengubah persamaaan B - C = 5.000 menjadi B = 5.000 + C. Lalu, kita eliminasi variabel B dengan cara substitusi seperti berikut.

A - B - 2C = 30.000

A - (5.000 + C) - 2C = 30.000

A - C - 2C = 30.000 + 5.000

A - 3C = 35.000

A + B + C = 200.000

A + (5.000 + C) + C = 200.000

A + 2C = 200.000 - 5.000

A + 2C = 195.000

Kemudian kita eliminasi variabel A untuk menentukan nilai variabel C seperti berikut.

A - 3C = 35.000

A + 2C = 195.000 _

-5C = -160.000

C = 32.000

Sehingga, kita dapat menentukan nilai variabel B dan variabel A sebagai berikut.

B = 5.000 + C

B = 5.000 + 32.000

B = 37.000

A + B + C = 200.000

A + 37.000 + 32.000 = 200.000

A = 200.000 - 37.000 - 32.000

A = 131.000
