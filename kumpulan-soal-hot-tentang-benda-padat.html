~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Diket : 
massa bola, m1 = 0,5 kg
kecepatan awal bola sebelum tumbukan, v1 = 10 m/s
massa tong, m2 = 5 kg
kecepatan awal tong sebelum tumbukan, v2 = 0
koef. Restitusi, e = 0,25

Tanya : 
Kecepatan bola dan tong setelah tumbukan, v1' = __ ? dan v2' = __?

Jawab : 

Step 1
---------
Hukum kekekalan momentum
m1.v1 + m2.v2 = m1.v1' + m2.v2'
0,5.(10) + 5.(0) = 0,5.v1' + 5.v2'   
5 = 0,5.v1' + 5.v2'
10 = v1' + 10v2' ..............Pers (1)

Step 2
---------
Koef. restitusi, e = - (v1' - v2') / (v1 - v2)
0,25 = - (v1' - v2') / (10 - 0)
0,25 = - (v1' - v2') / (10)
0,25. (10) = - (v1' - v2')
2,5 =  - (v1' - v2')
- 2,5 = v1' - v2' ..........Pers (2)

Step 3
---------
Eliminasi Pers (1) dan (2)

   10 = v1' + 10v2'
- 2,5 = v1' - v2'
------------------------------ minus
12,5 = 11.v2'
v2' = 12,5 / 11
v2' = 1,1364 m/s

dan masukkan ke pers (2) untuk mencari v1' = __?
- 2,5 = v1' - v2'
- 2,5 = v1' - 1,1364
v1' = - 2,5 + 1,1364 
v1' = -1,3636 m/s (tanda negatif artinya arah gerak bola ke kiri berlawanan dengan sebelum bertumbukan dengan tong)
jadi jawabannya adalah 
v1' = -1,3636 m/s dan v2' = 1,1364 m/s 

ATAU BISA JUGA GUNAKAN TRIK

Step 1
---------
C = (m1.v1 + m2.v2) / (m1 + m2)
C = (0,5.(10) + 5.(0)) / (0,5 + 5)
C = (5) / (5,5)
C = 0,909 

Step 2
---------
v1' = C + e(C - v1)
v1' = 0,909 + 0,25.(0,909 - 10)
v1' = 0,909 + 0,25.(-9,091)
v1' = 0,909 -2,27275
v1' = -1,36375 m/s

Step 3
---------
v2' = C + e(C - v2)
v2' = 0,909 + 0,25.(0,909 - 0)
v2' = 0,909 + 0,25.(0,909)
v2' = 0,909 + 0,22725
v2' = 1,13625 m/s
