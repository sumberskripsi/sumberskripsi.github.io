4. PP, bantu dong soal ulangan kelas 8 kurikulum 2013 ​


Jawaban soal 1:

11, 15, 19, 23, 27, 31, 35 memiliki rumus suku ke-n yaitu Un=4n + 7 dan untuk suku ke-22 hasilnya adalah 95, sementara jumlah 13 suku pertamanya adalah 455.

Jawaban soal 2:

3, 8, 13, 18 memiliki rumus suku ke-n yaitu Un=5n - 2 dan untuk suku ke-8 dan ke-30 adalah 38 dan 148. Nilai n ketika Un=28 adalah 6. Untuk jumlah 10 suku pertamanya adalah 255.

 

Penjelasan dengan langkah-langkah:

Rumus suku ke-n adalah Un = a + (n-1)b

Rumus jumlah suku ke-n adalah Sn = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

Keterangan:

a = suku pertama atau U1

b = beda atau selisih

Diketahui:

Soal 1: 11, 15, 19, 23, 27, 31, 35

Soal 2: 3, 8, 13, 18

Ditanya:

Soal 1:

a) dan b) kosong

c) Rumus suku ke-n

d) Suku ke-22

e) Jumlah 13 suku pertama

Soal 2:

a) Tentukan garisnya hingga ke U8

b) Tentukan rumus suku ke-n

c) Jika Un= 28, tentukan nillai n

d) Tentukan suku ke-30

e) Jumlah 10 suku pertama

Jawab:

Jawaban Soal 1:

11, 15, 19, 23, 27, 31, 35

dari soal didapat informasi bahwa:

a = 11

b = U2-U1 = 15-11 = 4

maka;

c) Un = a + (n-1)b

   Un = 11 + (n-1)4

   Un = 11 + 4n - 4

   Un = 4n + 7

d) Un   = 4n + 7

   U22 = 4(22) + 7

   U22 = 88 + 7

   U22 = 95 (suku ke-22 adalah 95)

e)  Sn  = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

    S13 = [tex]\frac{13}{2}[/tex] (2x11 + ((13-1)4))

    S13 =  [tex]\frac{13}{2}[/tex] (22 + 48)

    S13 =  [tex]\frac{13}{2}[/tex] (70)

    S13 = 455 (jumlah 13 suku pertamanya adalah 455)

Jawaban Soal 2:

3, 8, 13, 18

dari soal didapat informasi bahwa:

a = 3

b = U2-U1 = 8-3 = 5

maka;

a) Untuk melengkapi baris atau garis hingga U8 maka sederhananya tinggal menambahkannya dengan beda atau selisih 5, sehingga hasilnya:

3, 8, 13, 18, 23, 28, 33, 38

b) Un = a + (n-1)b

   Un = 3 + (n-1)5

   Un = 3 + 5n - 5

   Un = 5n - 2

c) Un = a + (n-1)b

   28 = 3 + (n-1)5

   28 = 3 + 5n - 5

   5n = 30

   n   = 6

d) Un   = 5n - 2

   U30 = 5(30) - 2

   U30 = 150 - 2

   U30 = 148 (suku ke-30 adalah 148)

e)  Sn  = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

    S10 = [tex]\frac{10}{2}[/tex] (2x3 + ((10-1)5))

    S10 =  [tex]\frac{10}{2}[/tex] (6 + 45)

    S10 =  5 (51)

    S10 = 255 (jumlah 10 suku pertamanya adalah 255)

Sifat-Sifat Bilangan Berpangkat
a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 x 33 = 39!
Jawab:
36 x 33 = ( 3 x 3 x 3 x 3 x 3 x 3) x (3 x 3 x 3)
6 faktor 3 faktor
= (3 x 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3)
(6 + 3) faktor
= 26 + 3
= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 : 33 = 33!
Jawab :
36 : 33 = 36
33
= (3 x 3 x 3 x 3 x 3 x 3)
3 x 3 x 3
= (3 x 3 x 3)
(6 - 3) faktor
= 36 - 3
= 33

c. Sifat Perpangkatan Bilangan Berpangkat
Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (43)2 = 46.
Jawab:
(43)2 = (43) x (43)
= (4 x 4 x 4) x (4 x 4 x 4)
3 faktor 3 faktor
= 4 x 4 x 4 x 4 x 4 x 4
(3 x 2) faktor
= 43 x 2 = 46

d. Sifat Perpangkatan pada Perkalian
Jika a, b bilangan bulat dan m bilangan bulat positifmaka (a x b)m = am x bm.
Contoh Soal=
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (32)3 = 33 x 23.
Jawab:
(32)3 = (3 x 2) x (3 x 2) x (3 x 2)
= (3 x 3 x 3) x (2 x 2 x 2)
3 faktor 3 faktor
= 33 x 23

e. Sifat Perpangkatan pada Pembagian.
Jika a, b bilangan bulat (b bukan 0) dan m bilangan bulat positif, maka (a : b)m = am : bm.
Contoh soal:
Dengan cara menuliskan dalam bentuk pembagian, tunjukkan bahwa (5 : 3)4 = 54 : 34.
Jawab:
(5 : 3)4 = (5/3)4
= (5/3) x (5/3) x (5/3) X (5/3)
= (5 x 5 x 5 x 5) = 54 = 54 : 53
(3 x 3 x 3 x 3) 34
