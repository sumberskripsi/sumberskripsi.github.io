6. Tolong Bantu! PTS KELAS 11 MATEMATIKA


1. Saldo akhir = Saldo awal + (Saldo awal * Bunga * Waktu)

Saldo awal = Rp 4.000.000,00

Bunga = 8% = 0,08 (dalam bentuk desimal)

Waktu = 3 tahun

Saldo akhir = 4.000.000 + (4.000.000 * 0,08 * 3)

= 4.000.000 + (960.000)

= 4.960.000

2. Modal akhir = Modal awal * (1 + Bunga)^Waktu

Modal awal = Rp 5.000.000,00

Bunga = 2% = 0,02 (dalam bentuk desimal)

Waktu = 8 bulan

Modal akhir = 5.000.000 * (1 + 0,02)^8

≈ 5.000.000 * (1,02)^8

≈ 5.000.000 * 1,16641

≈ 5.832.050.

3. Nilai Tunai = Modal / (1 + Bunga)^Waktu

Modal = Rp 100.000,00

Bunga = 5% = 0,05 (dalam bentuk desimal)

Waktu = 5 tahun

Nilai Tunai = 100.000 / (1 + 0,05)^5

= 100.000 / 1,27628

≈ 78.280,24

4. Bunga per caturwulan = (Bunga setahun) / (Jumlah caturwulan dalam setahun)

Bunga setahun = 5% = 0,05 (dalam bentuk desimal)

Jumlah caturwulan dalam setahun = 4

Bunga per caturwulan = 0,05 / 4

= 0,0125

Jumlah caturwulan untuk 1 tahun 3 bulan = 5

Besar bunga yang ditanggung = Pinjaman * Bunga * Jumlah caturwulan

= 10.000.000 * 0,0125 * 5

= 625.000

5. Kita akan menggunakan rumus bunga tunggal:

Saldo akhir = Saldo awal + (Saldo awal * Bunga * Waktu)

Saldo awal = Rp 4.000.000,00

Bunga = 4% = 0,04 (dalam bentuk desimal)

Saldo akhir = 4.000.000 + (4.000.000 * 0,04 * Waktu)

Kita diketahui bahwa saldo akhir = Rp 6.000.000,00

6. Tingkat bunga = ((Saldo akhir - Saldo awal) / Saldo awal) / Waktu

Saldo awal = Rp 10.000.000,00

Saldo akhir = Rp 12.000.000,00

Waktu = 9 bulan = 9/12 tahun

Tingkat bunga = ((12.000.000 - 10.000.000) / 10.000.000) / (9/12)

= ((2.000.000) / 10.000.000) / (9/12)

= 0,2 / 0,75

= 0,2667

Jadi, tingkat bunga tunggal tahunan atas pinjaman tersebut sekitar 26,67%.

7. Jumlah yang harus dibayar = Modal awal * (1 + Bunga)^Waktu

Modal awal = Rp 5.000.000,00

Bunga = 10% = 0,1 (dalam bentuk desimal)

Waktu = 3 tahun

Jumlah yang harus dibayar = 5.000.000 * (1 + 0,1)^3

= 5.000.000 * (1,1)^3

≈ 5.000.000 * 1,331

≈ 6.655.000

8. Besar bunga = Modal awal * Bunga * Waktu

Modal awal = Rp 2.000.000,00

Bunga = 6% = 0,06 (dalam bentuk desimal)

Waktu = 4 bulan = 4/12 tahun

Besar bunga = 2.000.000 * 0,06 * (4/12)

= 2.000.000 * 0,02

= 40.000

9. Waktu = (Jumlah akhir - Jumlah awal) / (Jumlah awal * Bunga)

Jumlah awal = Rp 800.000,00

Jumlah akhir = Rp 920.000,00

Bunga = 9% = 0,09 (dalam bentuk desimal)

Waktu = (920.000 - 800.000) / (800.000 * 0,09)

= 120.000 / 72.000

≈ 1,67

10. Modal awal = Jumlah akhir / (1 + Bunga)^Waktu

Jumlah akhir = Rp 10.868.000,00

Bunga = 8% = 0,08 (dalam bentuk desimal)

Waktu = 2 tahun

Modal awal = 10.868.000 / (1 + 0,08)^2

= 10.868.000 / 1,16641

≈ 9.323.013

8. contoh soal pts matematika kelas 7 semester 1yang bisa jawab aku kasih point lebih​


1. Cindy mempunyai uang sebesar Rp8.000,00 dan Janet mempunyai uang Rp2.000,00 lebihnya dari uang Cindy. Perbandingan uang Cindy dan Janet adalah ….

a. 2 : 8

b. 8 : 2

c. 4 : 5

d. 5 : 4

Jawaban: c

Pembahasan:

Misalkan uang Cindy = C

              uang Janet = J

Maka diperoleh:

C = 8.000

J = C + 2.000 = 8.000 + 2.000 = 10.000

Sehingga,

C : J = 8.000 : 10.000 = 4 : 5

Jadi, perbandingan uang Cindy dan Janet adalah 4 : 5
