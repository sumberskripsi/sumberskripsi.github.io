2. apa yang dimaksud cloud computing?dan jelaskan skema dan model bisnis cloud technology​


Cloud computing ialah gabungan pemanfaatan teknologi komputer dalam suatu jaringan pengembangan berbasis internet yang memiliki fungsi untuk menjalankan program maupun aplikasi melalui komputer-komputer yang terkoneksi pada waktu yang sama, akan tetapi tidak semua yang terkoneksi dengan internet menggunakan cloud computing.

Pembahasan  

Manfaat dari teknologi cloud computing :

Semua data tersimpan di server secara terpusat

Keamanan data

Fleksibilitas dan skabilitas yang tinggi

Investasi jangka panjang

Skema & model bisnis layanan cloud computing

IaaS (Infrastructure as a Service) : model komputasi awan dimana penyedia layanan memberikan jasa organisasi infrastruktur seperti server, hardware jaringan serta data stirage yang bisa digunakan konsumen.

PaaS (Platform as a Service) : model komputasi awan yang didesain agar memberikan developer sebuah platform yang memungkinkan developer tersebut untuk membuat dan mengembangkan aplikasi baru diatasnya.

Saas (Software as a Service) : model layanan komputasi awan yang ada dalam bentuk aplikasi yang diap untuk dipakai.

16. Jelaskan perbedaan antara Grid, Cloud dan Edge Computing!​


Jawaban :

Jelaskan perbedaan antara Grid, Cloud dan Edge Computing!

Apa itu Cloud Computing?

Cloud computing adalah pengiriman sumber daya komputasi sesuai permintaan termasuk server, penyimpanan, database, dan perangkat lunak melalui Internet, bukan server lokal atau komputer pribadi. Cloud adalah platform teknologi terdistribusi yang memanfaatkan inovasi teknologi canggih untuk menyediakan lingkungan yang sangat skalabel yang dapat digunakan dari jarak jauh oleh bisnis atau organisasi dalam banyak cara. Bisnis telah pindah ke cloud untuk skalabilitas, mobilitas, dan keamanan yang lebih baik. Cloud menyediakan mekanisme penyimpanan, pemrosesan, dan manajemen yang komprehensif. Bisnis dapat menghindari biaya di muka dan kompleksitas mempertahankan infrastruktur TI mereka sendiri, dan mereka hanya membayar untuk apa yang mereka gunakan dan ketika mereka menggunakannya. Beberapa penyedia layanan cloud utama termasuk Google, Oracle, Microsoft, IBM, Cisco, Verizon, dan Rackspace. Contoh komputasi awan adalah Microsoft Azure.

Apa itu Edge Computing??

Edge computing adalah paradigma komputasi terdistribusi yang membawa komputasi lebih dekat ke tepi jaringan, sebagai lawan dari struktur komputasi awan konvensional. Idenya adalah untuk memperluas komputasi awan ke cara yang lebih terdistribusi secara geografis di mana sumber daya komputasi, jaringan dan penyimpanan dapat didistribusikan di seluruh lokasi yang jauh lebih dekat dengan aplikasi pengguna akhir di mana ia dibutuhkan. Tujuan dari edge computing adalah bahwa komputasi harus terjadi pada kedekatan sumber data. Dalam komputasi tepi, hal-hal tidak hanya bertindak sebagai konsumen data tetapi juga berperan sebagai produsen data. Edge computing telah memainkan peran utama dalam mengatasi kompleksitas dalam komputasi awan - perangkat terdekat digunakan sebagai server untuk menyediakan layanan yang lebih baik. Ini berbagi gagasan tentang memindahkan komputasi lebih dekat ke perangkat tepi untuk mengatasi keterbatasan seperti bandwidth, latensi dan biaya tinggi dalam komputasi awan tradisional.

20. Uraikan perbedaan antara grid computing dan cloud computing !


Jawaban:

Perbedaan antara Grid Computing dan Cloud Computing

Teknologi yang terlibat dalam Grid Computing dan Cloud Computing

Komputasi grid adalah bentuk komputasi yang mengikuti arsitektur terdistribusi yang berarti satu tugas dipecah menjadi beberapa tugas yang lebih kecil melalui sistem terdistribusi yang melibatkan beberapa jaringan komputer. Cloud computing, di sisi lain, adalah kelas komputasi baru yang didasarkan pada teknologi jaringan di mana setiap pengguna cloud memiliki sumber daya pribadinya sendiri yang disediakan oleh penyedia layanan spesifik.

Terminologi Grid Computing dan Cloud Computing

Keduanya adalah teknologi komputasi berbasis jaringan yang memiliki karakteristik yang sama seperti pengumpulan sumber daya, namun keduanya sangat berbeda satu sama lain dalam hal arsitektur, model bisnis, interoperabilitas, dll. Komputasi grid adalah kumpulan sumber daya komputer dari berbagai lokasi untuk memproses suatu tugas tunggal. Grid bertindak sebagai sistem terdistribusi untuk berbagi sumber daya secara kolaboratif. Cloud computing, di sisi lain, adalah bentuk komputasi berdasarkan sumber daya tervirtualisasi yang terletak di beberapa lokasi dalam cluster.

Sumber Daya Komputasi dalam Komputasi Grid dan Cloud Computing

Komputasi grid didasarkan pada sistem terdistribusi yang berarti sumber daya komputasi didistribusikan di antara unit komputasi yang berbeda yang berlokasi di berbagai situs, negara, dan benua. Dalam komputasi awan, sumber daya komputasi dikelola secara terpusat yang terletak di beberapa server dalam kelompok di pusat data pribadi penyedia cloud.

Komunitas Peneliti

Dalam komputasi grid, sumber daya komputasi disediakan sebagai utilitas dengan grid sebagai platform komputasi yang didistribusikan secara geografis dan dikelompokkan dalam organisasi virtual dengan beberapa komunitas pengguna untuk memecahkan masalah skala besar melalui internet. Grid melibatkan lebih banyak sumber daya daripada hanya komputer dan jaringan. Cloud computing, di sisi lain, melibatkan kelompok umum administrator sistem yang mengelola seluruh domain.

Fungsi Grid Computing dan Cloud Computing

Fungsi utama dari komputasi grid adalah penjadwalan pekerjaan menggunakan semua jenis sumber daya komputasi di mana tugas dibagi menjadi beberapa sub-tugas independen dan setiap mesin pada grid ditugaskan dengan tugas. Setelah semua sub-tugas selesai, mereka dikirim kembali ke mesin utama yang menangani dan memproses semua tugas. Komputasi awan melibatkan pengumpulan sumber daya melalui pengelompokan sumber daya berdasarkan kebutuhan dari kelompok server.

Aplikasi Grid Computing dan Cloud Computing

Istilah "cloud" mengacu pada internet dalam komputasi awan dan secara keseluruhan itu berarti komputasi berbasis internet. Cloud mengelola data, persyaratan keamanan, antrian pekerjaan, dll. Dengan menghilangkan kebutuhan dan kompleksitas pembelian perangkat keras dan perangkat lunak yang diperlukan untuk membangun aplikasi yang akan dikirim sebagai layanan melalui cloud. Komputasi grid sebagian besar digunakan oleh penelitian akademik dan mampu menangani sejumlah besar pekerjaan berdurasi terbatas yang melibatkan volume data yang sangat besar.

Mengenal Apa itu Cloud Computing : Defenisi, Fungsi, dan Cara Kerja

Cloud Computing merupakan istilah yang sudah sering kita dengar. Namun, masih sedikit yang mengerti apa itu Definisi , Fungsi dan cara kerja Cloud Computing dalam kehidupan sehari – hari. Defisini Cloud Computing

Cloud Computing merupakan istilah dari Cloud diartikan sebagai internet dan Computing diartikan sebagai komputer. Definisi dari Cloud Computing adalah sebuah proses pengolahan daya komputasi melalui jaringan internet yang memiliki fungsi agar dapat menjalankan program melalui komputer yang telah terkoneksi satu sama lain pada waktu yang sama.

Cloud Computing merupakan sebuah teknologi yang menjadikan internet sebagai pusat server untuk mengelola data dan juga aplikasi pengguna. Cloud Computing memudahkan penggunanya untuk menjalankan program tanpa harus menginstall aplikasi terlebih dahulu dan memudahkan pengguna untuk mengakses data dan informasi melalui internet.

Google Drive adalah salah satu layanan penyimpanan berbasis cloud yang disediakan oleh Google. Dalam penggunaannya, pengguna dapat menyimpan berbagai jenis file seperti dokumen, gambar, video, dan lain sebagainya. Selain itu, Google Drive juga memungkinkan pengguna untuk berkolaborasi dengan pengguna lainnya dalam membuat dan mengedit dokumen secara bersama-sama.

Sementara itu, cloud computing adalah sebuah konsep teknologi yang memungkinkan akses ke sumber daya komputasi, seperti penyimpanan, aplikasi, dan infrastruktur jaringan melalui internet, sehingga pengguna tidak perlu memiliki perangkat keras dan infrastruktur yang mahal. Cloud computing memungkinkan pengguna untuk memperoleh sumber daya IT sesuai dengan kebutuhan mereka dan membayar hanya untuk sumber daya yang mereka gunakan.

Kesimpulan dari Google Drive dan cloud computing adalah bahwa keduanya merupakan teknologi yang berbasis cloud. Google Drive adalah salah satu contoh layanan penyimpanan cloud yang dapat diakses melalui internet, sedangkan cloud computing merupakan sebuah konsep teknologi yang memungkinkan akses ke berbagai sumber daya IT melalui internet. Keduanya memberikan keuntungan bagi pengguna, seperti kemudahan akses dan kolaborasi, penghematan biaya, dan fleksibilitas dalam penggunaan sumber daya IT.

40. 1. Jelaskan apa yang menjadi perbedaan mendasar antara grid computing dengan cloud computing!


Jawaban:

Cloud Computing adalah Untuk Berorientasi Layanan sedangkan Grid Computing adalah untuk Berorientasi Aplikasi.

Komputasi grid digunakan untuk membangun superkomputer virtual menggunakan perangkat middler untuk mencapai tugas bersama yang dapat dibagi di antara beberapa sumber daya.

Penjelasan:

Cloud bersifat interaktif karena Anda bisa mendapatkan sumber daya sesuai permintaan melalui layanan mandiri. Kode yang Anda jalankan di VM di cloud, seperti server web Apache, dapat server klien secara interaktif.

Grid adalah kumpulan di mana Anda mengirimkan pekerjaan ke antrian pekerjaan setelah mendapatkan kredensial dari beberapa otoritas HPC untuk melakukannya. Kode yang Anda jalankan di grid menunggu dalam antrian itu sampai ada sumber daya yang cukup untuk menjalankannya.
