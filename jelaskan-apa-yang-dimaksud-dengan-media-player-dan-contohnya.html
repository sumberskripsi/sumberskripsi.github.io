6. Bagaimanakah cara mengaktifkan Program Windows Media Player? Jelaskan langkah-langkahnya!


Jawaban:

cara mengaktifkan Program Windows Media Player? Jelaskan langkah-langkahnya!

Mulai Editor Registri.

Cari kunci berikut dalam registri:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft

Pada menu Edit, arahkan ke Baru, lalu klik Kunci.

Ketik WindowsMediaPlayer, lalu tekan ENTER.

Dengan tombol WindowsMediaPlayer yang dipilih, pada Edit menu, arahkan ke baru, dan kemudian klik Nilai DWORD.

Ketik DisableAutoUpdate, lalu tekan ENTER.

Pada Edit menu, klik Ubah.

Ketik 0 (atau menghapus 1), kemudian klik OK.

Tutup Editor Registri.

Mengaktifkan Windows Media Player untuk secara berkala memeriksa pembaruan dengan menggunakan tataan kebijakan grup

Untuk mengaktifkan Windows Media Player untuk memeriksa pembaruan secara otomatis dengan menggunakan Setelan Kebijakan Grup, ikuti langkah-langkah berikut:

Di grup kebijakan konsol manajemen Microsoft (MMC), di bawah Konfigurasi Komputer luaskan Pola Dasar Administratif, luaskan Komponen Windows, lalu klik Windows Media Player.

Dalam daftar kebijakan yang tersedia, klik dua kali Mencegah Pembaruan Otomatis.

Klik Dinonaktifkan atau klik Tidak Dikonfigurasi, lalu klik OKE.

Tutup Kebijakan Grup MMC.

19. bagaimana cara agar windows media player dapat memutar video


klik kanan, lalu klik 'open with', trus pilih 'windows media player'- Langkah pertama, Anda sediakan Aplikasi K-lite Codec versi terbaru. Aplikasi K-lite Codec kita gunakan karena aplikasi ini memiliki file – file codec yang sangat lengkap. Anda Dapat mengunduhnya disini K-Lite Codec
- Pada saat file Assosiasi, centang Windows Media Player agar Windows Media Player dapat bekerja sama dalam menggunakan codec bersamaan. Codec merupakan suatu library yang berfungsi untuk memutar / membuka / membaca suatu format video atau music.
- Setelah itu, Anda akan disuruh untuk memilih tipe file baik Video atau pun Audio yang akan diputar pada Media Player Classic. Untuk itu, Anda klik button Select All Video dan Klik button Select All Audio. Ini berfungsi agar Media Player Classic dapat menjalankan tipe file tersebut.
- Langkah selanjutnya, Anda Akan disuruh untuk memilih tipe file yang akan diputar oleh Windows Media Player. Hal ini hampir sama dengan langkah diatas yaitu Anda harus klik button Select All Video dan Klik button Select All Audio. Ini berfungsi agar Media Player Player dapat menjalankan tipe file tersebut.
- Dan langkah selanjutnya. Anda cukup klik button Next hingga selesai.
Gampang kan caranya. Nah dengan beginikan Anda tidak perlu menginstal aplikasi yang lain lagi. Cukup menjalankan Windows media player sebagai player Video dan Music Anda semua.

38. Makna logo internet explorer dan Windows media player


Microsoft Windows

Pada awalnya, jika melihat logo Windows, yang terpikir tentu gambar bendera yang melambai. Padahal, logo windows adalah gambar jendela (dalam bahasa Inggris, jendela=window). Menurut pihak Microsoft, jendela adalah sebuah kiasan yang sangat baik untuk menggambarkan teknologi komputasi karena pada dasarnya, Microsoft dan Windows ingin ‘memberikan’ teknologi ke tangan para penggunanya dan mengajak user melakukan berbagai hal dengan teknologi tersebut sesuai perspektif masing-masing user.
