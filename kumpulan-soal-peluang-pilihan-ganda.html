Penjelasan dengan langkah-langkah:

misalkan

a = angka

g = gambar

ruang sampel pelemparan 3 koin adalah

{aaa, aag, aga, agg, gaa, gag, gga, ggg}

banyak anggota = 8

himpunan kejadian munculnya 3 angka

= {aaa}

banyak anggota = 1

himpunan kejadian minimal 2 gambar

= {agg, gag, gga, ggg}

banyak anggota = 4

a. peluang muncul 3 angka

= banyak kejadian munculnya 3 angka / banyak anggota ruang sampel

= 1/8

b. peluang muncul minimal 2 gambar

= banyak kejadian muncul minima 2 gambar / banyak anggota ruang sampel

= 4 / 8

= 1/2
