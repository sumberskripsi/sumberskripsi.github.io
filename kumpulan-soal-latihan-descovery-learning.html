32. Perbedaan active learning dan basic learning?​


Jawaban:

Perbedaan antara active learning dan basic learning adalah sebagai berikut:

1. Pendekatan Pembelajaran:

- Basic Learning: Basic learning mengadopsi pendekatan pembelajaran tradisional di mana guru berperan sebagai pemegang pengetahuan yang menyampaikan informasi kepada siswa. Siswa lebih pasif dalam proses pembelajaran dan cenderung menjadi penerima informasi.

- Active Learning: Active learning mendorong keterlibatan aktif siswa dalam proses pembelajaran. Siswa berperan sebagai pembelajar aktif yang secara aktif terlibat dalam kegiatan seperti diskusi, kolaborasi, pemecahan masalah, eksperimen, dan proyek.

2. Posisi Guru:

- Basic Learning: Dalam basic learning, guru berperan sebagai sumber pengetahuan utama dan pemandu pembelajaran. Mereka memberikan informasi, menjelaskan konsep, dan memberikan tugas kepada siswa.

- Active Learning: Dalam active learning, guru berperan sebagai fasilitator pembelajaran. Mereka memfasilitasi diskusi, memberikan bimbingan, memberikan umpan balik, dan mengarahkan siswa dalam mengembangkan pemahaman mereka sendiri.

3. Fokus Pembelajaran:

- Basic Learning: Basic learning cenderung lebih fokus pada penerimaan dan penyerapan informasi oleh siswa. Tujuannya adalah untuk mentransfer pengetahuan dari guru ke siswa.

- Active Learning: Active learning fokus pada pemberdayaan siswa dalam mengembangkan pemahaman, keterampilan, dan penerapan konsep. Tujuannya adalah untuk melibatkan siswa secara aktif dalam proses pembelajaran sehingga mereka dapat mengonstruksi pengetahuan mereka sendiri.

4. Interaksi Siswa:

- Basic Learning: Interaksi siswa dalam basic learning cenderung terbatas pada guru. Siswa mungkin tidak banyak berinteraksi dengan sesama siswa.

- Active Learning: Active learning mendorong interaksi siswa dengan sesama siswa. Siswa berdiskusi, berkolaborasi, dan berbagi ide untuk memperluas pemahaman mereka.

Dalam kesimpulannya, active learning lebih menekankan keterlibatan aktif siswa dalam proses pembelajaran dan memungkinkan mereka untuk mengembangkan pemahaman yang lebih mendalam melalui interaksi, eksplorasi, dan pemecahan masalah. Sementara basic learning lebih cenderung pada pendekatan tradisional di mana guru berperan sebagai pemegang pengetahuan utama dan siswa lebih pasif sebagai penerima informasi.

34. Apa hubungan e-learning dengan learning Manajemen system dan learning content manajemen system

Sistem ini disebut ‘’Learning Content Management Systems’’ atau LCMS. Meskipun perbedaan ini, LMS sering digunakan untuk merujuk antara LMS dan LCMS, meskipun LCMS merupakan pengembangan lebih lanjut dari LMS.
