13. Seorang siswa melakukan percobaan difraksi cahaya dengan menggunakan kisi difraksi 400 garis/mm. Ia mendapatkan pita terang pertama untuk suatu cahaya monokromatik berjarak 1.2 cm dari pusat terang. Panjang gelombang cahaya yang ia gunakan adalah​


diket:

k = 400 garis/mm = 400.000 garis/m

jarak antar celah (d) = 1/k

= 1/400.000 = 0,25 x 10⁻⁵ m = 25 x 10⁻⁷ m

jarak pita terang pertama (p) = 1,2 cm = 1,2 x 10⁻² m

pita terang pertama ---> n = 1, l = 1

ditanya:

panjang gelombang (λ)...?

jawab:

p d / l = n λ

λ = p d /  n l

  = (1,2 x 10⁻² m x 25 x 10⁻⁷ m) / (1 x 1)

  = 30 x 10⁻²⁻⁷ m

  = 30 x 10⁻⁹ m

Jadi, panjang gelombang cahaya yang ia gunakan adalah​ 30 x 10⁻⁹ m.

15. Cahaya monokromatis melalui kisi difraksi. Lebar kisi 0,1 mm,jarak antara kisi dengan layar 1m jika jarak antar 2 garis terang adalah 1,5mm. Berapa panjang gelombang cahaya tersebut


Difraksi cahaya merupakan peristiwa pelenturan cahaya karena melalui celah kecil. Pada difraksi kisi, cahaya dilenturkan melalui celah- celah kecil berjumlah banyak sehingga akan tampak pola terang denga orde tertentu.


Dari soal di atas dapat diketahui bahwa:

- Terjadi difraksi kisi/ celah majemuk

- jarak celah layar --> l = 1 m

- lebar kisi -> d = 0,1 mm =  10^-4 m

- jarak dua terang --> p = 1,5 mm = 1,5. 10^-3 m

 

Sedangkan yang menjadi permasalahan pada soal adalah panjang gelombang.

 

Mula – mula kita cari n untuk dua terang berdekatan yaitu n = 1.

 

Untuk mencari panjang gelombang digunakan rumus berikut:

nλ = d.p/l

1. λ  = 10^-4. 1,5. 10^-3 / 1

λ = 1,5. 10^-7 m = 150 nm

 

Berdasarkan uraian di atas, didapat bahwa panjang gelombang yang digunakan 150 nm. 

 

 

PEMBAHASAN LEBIH LANJUT :

 

Persamaan umum yang digunakan pada interferensi dan difraksi cahaya yaitu:

nλ = d.p/l

 

atau

nλ = d sin θ

 

keterangan:

n = orde terang/ gelap

λ = panjang gelombang

d = jarak celah

p = jarak antar pola

l = jarak layar ke celah

θ = sudut deviasi

 

Ketentuan n pada kisi difraksi:

Pola terang: n = 3/2, 5/2, 7/2, 9/2, dst.

Pola gelap: n = 1, 2, 3, 4, dst..

 

Pada kisi difraksi terdapat jumlah celah/ goresan tiap meter yang disebut N

N = 1/d

 

Untuk memantapkan kalian memahami soal ini, coba simak soal seperti di bawah ini:

Cahaya monokromatik dengan panjang gelombang 600 nm datang tegak lurus pada kisi yang mempunyai lebar celah 2 μm. Tentukan sudut difraksi untuk garis terang pertama!

 

Untuk mencari sudut difraksi garis terang 1 (n = 1) gunakan rumus:
nλ = d sin θ
1. 600. 10^-9 = 2. 10^-6. sin θ
0, 3 = sin θ
θ = 17,5°

PEMBAHASAN

Difraksi cahaya adalah peristiwa pembelokan arah rambat cahaya karena melewati celah sempit.

Kisi difraksi adalah peristiwa difraksi saat cahaya melewati celah-celah sempit yang sangat halus dan banyak.

d. sin Ф = n . λ

dimana:

d = jarak antar celah sempit ( m )

Ф = sudut pembelokan cahaya ( derajat )

n = orde

λ = panjang gelombang ( m )

Jika cahaya mengenai suatu celah tunggal maka berlaku rumus yang mirip yakni :

d. sin Ф = n . λ

dimana:

d = lebar celah tunggal ( m )

Ф = sudut pembelokan cahaya ( derajat )

n = orde

λ = panjang gelombang ( m )

Mari sekarang kita mencoba menyelesaikan soalnya.

Interferensi Konstruktif dimana berkas-berkas cahaya bergabung dan saling menguatkan sehingga terbentuk pita terang di layarInterferensi Destruktif dimana berkas-berkas cahaya bergabung dan saling melemahkan sehingga terbentuk pita gelap di layar
