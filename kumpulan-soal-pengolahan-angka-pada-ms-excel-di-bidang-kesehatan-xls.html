12. Ekstensi File yang menggunakan Macro VBA pada Ms. Excel adalah * a. .xls b. ppt c. .xlsm d. docm​


Jawaban: C

Penjelasan:

Extensi file dari dokumen Microsoft Excel yang menggunakan Macro VBA adalah xlsm, dengan xls menunjukkan bahwa file tersebut adalah dokumen Microsoft Excel dan m menunjukkan bahwa dokumen Microsoft Excel tersebut menggunakan Macro VBA.

Untuk pilihan lainnya:

A. xls -> Merupakan file dokumen Microsoft Excel biasa

B. ppt -> Merupakan file dokumen Microsoft Powerpoint

D. docm -> Merupakan file dokumen Microsoft Word yang menggunakan Macro VBA

Jawaban:

c. .xlsm

Penjelasan:

xlsm berarti Microsoft Excel Macro-Enable Worksheet. Keterangan ini bisa dilihat dari properties file.

Perhatikan gambar berikut ini (ditunjukkan tanda panah).


13. cara memformat tampilan angka pada ms excel


1. klik sel pada excel yang ingin difotmat tampilan angkanya
2. klik kanan
3. klik format cells
4. klik tab. Number
- pilih Number= untuk memformat tampilan angka biasa
- pilih Currency= bila ingin memformat tampilan angka disertai mata uang
5. Klik OK

20. File hasil olahan ms. excel disebut dengan istilah ….


Jawaban:

xlsx atau excel stylesheet

Penjelasan:

File dengan ekstensi.xlsx merupakan file Microsoft Excel Open XML Spreadsheet (XLSX) yang dibuat oleh Microsoft Excel. Jadi, kamu juga dapat membuka format bentuk file tersebut selain dari Excel, kamu juga bisa membukanya pada aplikasi spreadsheet lainnya, termasuk Apple Numbers, Google Docs, dan OpenOffice. File .xlsx biasanya terdiri dari tabel-tabel yang berisi data-data yang sudah/akan diolah.

34. Ms. Excel merupakan software pengolahan angka dan ...a   katab  gambarc   tabeld   animasi​


C. Tabel

Pembahasan • • •

Microsoft Excel Merupakan Software pengolahan angka dan Tabel.

Microsoft Excel Adalah salah satu bagian dari Microsoft Office System, karena Microsoft Excel memiliki fungsi sebagai pengolahan data seperti mencari rata rata suatu angka, pertambahan secara cepat, pengurangan, perkalian, pembagian, dsb. Serta fungsi Microsoft Excel juga sebagai Pengolah atau pembuat tabel.

Dalam Microsoft Excel telah disediakan kotak kotak yang bisa kita jadikan tabel dari pengolahan data yang telah kita buat sebelum nya, sehingga Microsoft Excel sangat sering digunakan oleh pekerja, mahasiswa, bahkan murid untuk bisa memudahkan pengolahan data yang ingin dibuat.
