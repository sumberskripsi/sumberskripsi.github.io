21. Jelaskan model jaringan thin client - thick server


Model jaringan thin client dan thick server

Thin client dan thick server merupakakn sebuha model jaringan dimana client menjalankan fungsi hanya sebagai penyaji aplikasi dan data yang diakses dan dikelola server. Sehingga beban yang paling berat dirasakan oleh server. MOdel ini biasa disebut dengan thin client.

Pembahasan

Thin client adalah sebuah komputer atau program komputer yang sangat bergantung pada beberapa komputer lain (server) untuk memenuhi peran tradisional komputasi.

Thick server adalah sebuah komputer (klien) dalam arsitektur client-server atau jaringan yang biasanya menyediakan fungsionalitas yang kaya independen dari server pusat. Kelebihan pada model jaringan ini adalah apabila digunakan pada lab komputer client tidak perlu menginstall aplikasi yang akan digunakan.


Selain itu ada beberapa kelebihan dari model jaringan thin client:

1. Longer hardware lifecycle

2. Investasi hardware jauh lebih murah

3. Maintenance

4. Manajemen desktop

5. Update lebih mudah dan lebih murah

6. Keamanan data

Istilah Client Server merujuk kepada model populer untuk jaringan komputer yang memanfaatkan perangkat hardware pada klien dan server, dengan fungsi spesifik masing-masing. Model Client Server dapat digunakan pada Internet dan juga pada LAN (Local Area Network).  

Jaringan Client Server mulai populer sejak komputer personal menjadi alternatif umum daripada komputer mainframe. Komputer Client pada umumnya merupakan komputer personal yang memiliki aplikasi perangkat lunak jaringan, yang meminta dan menerima informasi melalui jaringan. Perangkat mobile, begitu juga dengan komputer dekstop, dapat berfungsi sebagai komputer Klien.

Komputer Server biasanya menyimpan file dan database, yang meliputi aplikasi kompleks seperti site Web. Komputer Server biasanya memiliki prosesor berkekuatan besar, memori yang besar, dan disk drive yang lebih besar daripada komputer Klien.

Model Client Server mengatur trafik jaringan melalui aplikasi klien dan juga melalui perangkat. Client mengirim pesan dan membaut request ke server. Server akan merespon klien dengan menjalankan setipa request dan mengembalikan hasil. Satu server dapat mendukung banyak klien, dan multiple server dapat saling dihubungkan untuk menangani proses dalam jumlah besar, pada saat jumlah klien meningkat.

Komputer klien dan komputer server biasanya merupakan 2 unit perangkat keras yang terpisah, dimana masing-masing unit dikostumisasi untuk tujuannya masing-masing.

Beberapa contoh aplikasi populer yang menggunakan model Client Server adalah email, FTP dan Web Service. Setiap dari klien memiliki tampilan antar muka ( baik gambar maupun teks ) dan aplikasi klien yang memungkinkan pengguna untuk terhubung ke server. Untuk email dan FTP, pengguna memasukkan nama komputer (atau alamat IP) kedalam interface untuk mengatur koneksi ke server.

28. jaringan peer to peer adalah jaringan di mana setiap komputer dapat berperan sebagai server sekaligus client. Kapan sebuah komputer akan menjadi server dan kapan menjadi client?


Jaringan peer to peer adalah jaringan di mana setiap komputer dapat berperan sebagai server sekaligus client. Kapan sebuah komputer akan menjadi server dan kapan menjadi client? Jaringan peer to peer merupakan jaringan yang lebih dikenal dengan jaringan torrent. Karena pada file-file yang berbagi menggunakan sistem torrent biasanya menggunakan jaringan peer to peer untuk proses pembagian filenya.

Pada jaringan peer to peer semua komputer akan terhubung dan semua komputer dapat menjadi server dan menjadi client. Ketika salah satu komputer meminta suatu file, maka komputer lain akan bertindak sebagai servernya, dan komputer yang meminta file tersebut akan menjadi klien.

Demikian pembahasan mengenai jaringan peer to peer dimana jaringan peer to peer merupakan jaringan yang terdiri dari beberapa komputer, namun seiring perkembangan jaman, jaringan peer to peer dikenal sebagai jaringan torrent. Pada jaringan peer to peer, semua komputer bisa menjadi server dan bisa menjadi client.
