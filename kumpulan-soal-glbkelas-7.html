9. Di jawabyah plisss soalnya udah mau di kumpul nanti Jam 7​


Jawaban:

A = 2

K. segitiga= 24 cm

Penjelasan dengan langkah-langkah:

(3A+4)² = (3A+2)² + (A+4)²

9A²+12A+16 = (9A²+12A+4)+(A²+8A+16)

9A²+12A+16 = 10A²+20A+20

9A²-10A²+24A-20A+16-20 = 0

-A²+4A-4 = 0

A²-4A+4 = 0

(A-2)² = 0

A-2 = 0

A = 2

Keliling = (3A+2) + (A+4) + (3A+4)

= (3(2)+2) + (2+4) + (3(2)+4)

= (6+2)+(2+4)+(6+4)

= 8+6+10

= 24 cm
